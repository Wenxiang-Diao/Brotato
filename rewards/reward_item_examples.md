## 9. MVP 奖励数量建议

MVP 初版建议至少实现：

| 类别 | 数量 |
|---|---:|
| 通用成长 | 8-12 |
| 状态核心 | 12-18 |
| 武器强化 | 8-12 |
| 生存修正 | 6-10 |
| 风险奖励 | 8-12 |
| 资源经济 | 6-8 |
| 角色技能 | 4-8 |
| Boss 专属 | 3-6 |

总量建议：**60-80 个奖励**，足够支撑 MVP 随机池与多流派构筑。
