# 💎 奖励品质与稀有度规则

> 目标：让品质不仅代表数值强度，也代表机制复杂度、风险程度和构筑价值。

---

## 1. 品质分级

| 品质 | 定位 | 典型作用 |
|---|---|---|
| 蓝 | 基础成长 | 小幅数值、基础状态强化、低成本补强 |
| 紫 | 构筑雏形 | 明确流派方向，提供中等机制收益 |
| 粉 | 构筑核心 | 形成状态联动或改变武器节奏 |
| 红 | 强构筑转折 | 大幅强化某流派，但通常有条件 |
| 金 | 稀有机制核心 | 改变玩法循环，必须有限制或代价 |

---

## 2. 蓝色奖励

### 设计定位

蓝色奖励用于基础补强，不能成为决定性构筑核心。

### 示例

| 奖励 | 效果 |
|---|---|
| 攻击校准 | 攻击力 +6% |
| 轻量扳机 | 攻速 +5% |
| 小型晶盾 | 最大生命 +8 |
| 微燃粉尘 | 灼烧持续时间 +0.4 秒 |
| 导电铜线 | 感电附着概率 +4% |

---

## 3. 紫色奖励

### 设计定位

紫色奖励开始引导构筑方向。

### 示例

| 奖励 | 效果 |
|---|---|
| 连锁导线 | 感电连锁跳转 +1，但伤害衰减 10% |
| 毒雾瓶 | 中毒目标死亡时向周围扩散 1 层中毒 |
| 冰裂纹 | 冻结结束时造成一次小范围伤害 |
| 标记镜片 | 标记目标受到暴击率 +8% |
| 破甲楔片 | 对破甲目标攻击速度 +6% |

---

## 4. 粉色奖励

### 设计定位

粉色奖励是流派成型的重要节点。

### 示例

| 奖励 | 效果 | 限制 |
|---|---|---|
| 爆燃链式反应 | 灼烧 + 中毒触发爆裂范围 +25% | 爆裂冷却 2 秒 |
| 雷鸣标靶 | 感电命中标记目标时额外跳转 2 次 | 每目标 3 秒内只触发一次 |
| 冰核破碎 | 冻结目标受到重击时产生破碎伤害 | Boss 只承受 40% 控制效果 |
| 结晶穿透 | 破甲层数达到 3 时下次攻击穿透 | 触发后清空破甲层数 |
| 饥渴回路 | 击杀状态目标回复 1% 最大生命 | 每秒最多触发 2 次 |

---

## 5. 红色奖励

### 设计定位

红色奖励是高价值构筑转折，必须具备限制。

### 示例

| 奖励 | 效果 | 代价 / 限制 |
|---|---|---|
| 过载燃心 | 过载伤害 +60% | 每次过载后自身获得 1 层灼烧压力 |
| 雷暴扩音器 | 感电链跳无视距离一次 | 连锁伤害衰减增加 |
| 霜域核心 | 冻结敌人死亡时留下冰域 | 移动速度 -5% |
| 腐殖王冠 | 中毒层数上限 +1 | 非中毒伤害 -8% |
| 破界楔石 | 破甲可对 Boss 额外生效 | 普通怪清怪效率降低 |

---

## 6. 金色奖励

### 设计定位

金色奖励是稀有玩法核心，应该让整局策略发生明显变化。

金色奖励必须满足至少一条限制：

1. 绑定 debuff
2. 需要特定状态组合
3. 有触发冷却
4. 牺牲某类属性
5. 限定某类武器
6. 只能在 Boss / 事件 / 风险节点获得

### 示例

| 奖励 | 效果 | 限制 |
|---|---|---|
| 星链反应炉 | 每次三状态反应后，下一次武器攻击触发额外元素爆发 | 每 6 秒最多触发一次，获得 1 个轻度 debuff |
| 古树逆枝 | 每拥有 1 个 debuff，所有状态伤害 +8% | 最大生命 -12% |
| 无限导能环 | 感电连锁不再降低跳转次数 | 每次连锁后伤害衰减更高 |
| 凝流静界 | 进入低血时冻结周围敌人并生成护盾 | 冷却 45 秒，移动速度降低 |
| 赤焰终幕 | 灼烧目标死亡时引爆全部层数 | 灼烧无法自然刷新持续时间 |

---

## 7. 品质与风险关系

| 品质 | 是否允许无代价 | 风险要求 |
|---|---|---|
| 蓝 | 可以 | 无 |
| 紫 | 可以 | 通常无 |
| 粉 | 可以，但需触发条件 | 小限制 |
| 红 | 不建议无代价 | 条件、冷却、负面属性或轻中度 debuff |
| 金 | 不允许无代价 | 必须存在明确代价 |

---

## 8. 品质与 UI 表达

| 品质 | UI 表现 |
|---|---|
| 蓝 | 简洁边框，突出基础效果 |
| 紫 | 增加标签提示，强调流派方向 |
| 粉 | 特效边框，显示状态联动 |
| 红 | 风险标识，提示代价 |
| 金 | 独立动画，必须显示“核心机制 / 代价” |

---

## 9. 品质平衡原则

1. 品质越高，机制越强，但越不能泛用。
2. 金色奖励应鼓励改变打法，而不是单纯数值碾压。
3. 红 / 金奖励需要在说明中清晰展示代价。
4. 蓝 / 紫奖励保证基础体验，防止前期空转。
5. 粉色奖励是 MVP 中最重要的流派成型层级。
