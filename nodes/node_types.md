# 节点类型详细设计

---

## 1. 普通战斗节点

### 功能定位

基础推进节点，用于提供经验、金币和低中品质构筑材料。

### 战斗内容

- 普通怪为主；
- 少量特殊机制怪；
- 第 2 章后可出现小型精英单位；
- 战斗时长控制在 60-120 秒。

### 奖励

| 奖励 | 说明 |
|---|---|
| 经验 | 稳定获得 |
| 金币 | 稳定获得 |
| 普通武器 / buff | 低概率 |
| 状态强化 | 低概率出现 |

### 风险

- 基础血量消耗；
- 如果玩家 build 未成型，可能被怪物密度压制；
- 后期普通节点会加入环境压力。

### 适合选择场景

- 需要稳定升级；
- 当前血量较低，不适合挑战精英；
- 缺少金币或经验。

---

## 2. 精英战斗节点

### 功能定位

高风险高收益节点，用于检验当前 build 强度。

### 战斗内容

- 1 个精英怪 + 多波普通怪；
- 精英带 1-2 个词缀；
- 战斗时长控制在 120-180 秒。

### 精英词缀示例

| 词缀 | 效果 |
|---|---|
| 厚壳 | 受到的单次伤害上限降低 |
| 反涌 | 受到状态后短时间强化移动速度 |
| 聚怪 | 周期性召唤小怪 |
| 抗冻 | 冻结时间降低 |
| 雷导 | 感电会在敌方单位间传导 |

### 奖励

- 稀有武器或武器升阶材料；
- 稀有 buff；
- 水晶碎片；
- 高概率状态强化；
- 少量角色羁绊点。

### 风险

- 高血量消耗；
- 精英词缀可能克制当前 build；
- 失败后资源损失更大。

---

## 3. Boss 节点

### 功能定位

章节检验节点，判断玩家当前构筑是否达标。

### 规则

- 第 6、12、18 层固定 Boss；
- Boss 至少 2 个阶段；
- Boss 拥有部分状态抗性；
- Boss 不完全免疫状态，但控制效果衰减。

### 奖励

| Boss | 奖励方向 |
|---|---|
| 第 1 章 Boss | 核心 build 补强，解锁中级奖励池 |
| 第 2 章 Boss | 高品质武器、稀有 buff、水晶碎片 |
| 最终 Boss | 大量局外资源、角色羁绊、图鉴解锁 |

---

## 4. 事件节点

### 功能定位

提供非战斗决策，强化 Roguelite 随机性。

### 事件结构

每个事件至少包含 2-3 个选项：

- 稳定收益；
- 高风险收益；
- 修正或放弃收益；
- 特殊 build 选项。

### 可能奖励

- 移除 debuff；
- 获得特殊 buff；
- 获得武器；
- 改变抽奖权重；
- 获得水晶碎片；
- 触发隐藏路线。

### 风险

- 获得 debuff；
- 损失血量；
- 损失金币；
- 下一场战斗敌人强化；
- 抽奖池被污染。

---

## 5. 商店节点

### 功能定位

资源转换和 build 修正节点。

### 可购买内容

| 商品 | 说明 |
|---|---|
| 武器 | 随机 3-5 个 |
| 状态强化 | 强化某一状态附着或反应 |
| 功能道具 | 回血、复活、护盾 |
| debuff 修正 | 移除或降低负面效果 |
| 重抽券 | 用于后续抽奖节点 |
| 水晶碎片 | 高价低频商品 |

### 特殊规则

- 商店价格随章节提升；
- 连续经过商店会降低商品稀有度；
- 风险型角色可购买“诅咒折扣”商品。

---

## 6. 抽奖节点

### 功能定位

核心构筑推进节点。

### 基础规则

进入后从 3 个奖励中选择 1 个：

- 武器；
- buff；
- 状态强化；
- 元素倾向强化；
- 特殊构筑组件。

### 风险选项

抽奖节点中有概率出现“高光奖励”：

> 获得高品质奖励，但附带一个 debuff。

例如：

```text
获得紫色感电强化，但下一场战斗敌人移动速度 +10%。
```

---

## 7. 休息节点

### 功能定位

恢复与修正节点。

### 可选行为

玩家只能选择一项：

| 行为 | 效果 |
|---|---|
| 休整 | 恢复 35% 最大生命 |
| 净化 | 移除 1 个普通 debuff |
| 精炼 | 提升 1 个武器或状态的稳定性 |
| 冥想 | 提高下一次抽奖的稀有度权重 |

### 设计目的

休息节点本身不直接提高战斗力，主要用于降低风险。

---

## 8. 宝藏 / 水晶节点

### 功能定位

局外成长和特殊资源节点。

### 奖励

- 水晶碎片；
- 角色羁绊点；
- 图鉴素材；
- 稀有但不立即生效的成长资源。

### 风险

- 当前局战斗收益较低；
- 可能错过武器或状态强化；
- 高难度下可能附带守卫战斗。

---

## 9. 风险节点

### 功能定位

服务高风险高收益构筑。

### 规则

玩家进入前会看到风险标签：

| 标签 | 说明 |
|---|---|
| 诅咒 | 必定获得 debuff |
| 强敌 | 战斗敌人强化 |
| 限时 | 限时完成才获得完整奖励 |
| 失衡 | 玩家某项属性暂时降低 |

### 奖励

- 高品质武器；
- 稀有状态反应强化；
- 大量水晶碎片；
- 强力 buff；
- 特殊隐藏事件入口。
