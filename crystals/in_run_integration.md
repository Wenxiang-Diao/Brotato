# 🎮 水晶系统与局内系统联动

---

## 1. 与抽奖 / 奖励系统联动

水晶最重要的局内影响是改变奖励池权重。

### 1.1 权重原则

水晶不直接保证某个奖励出现，而是提高相关标签权重。

示例：

```text
导能分支投资较高
→ 感电、标记、连锁、高频触发类奖励权重提升
→ 玩家更容易组出导能 build
→ 但仍需要局内选择和运气
```

### 1.2 权重上限

单一分支对奖励池的总权重提升建议不超过 15%。

避免出现：

- 玩家开局必定成型
- 每局奖励过于重复
- 水晶分支压倒角色和武器选择

### 1.3 保底联动

星核节点可影响保底类型。

例如：

- 连续 3 次未出现当前主构筑标签奖励时，提高相关权重。
- 当前角色元素倾向与水晶分支一致时，保底计数略快。

## 2. 与 Debuff 系统联动

水晶不能取消 Debuff 系统的风险定位，只能提供管理工具。

### 2.1 可提供能力

- 提高修正窗口质量
- 增加 Debuff 替换选项
- 延迟某个 Debuff 生效
- 将轻微 Debuff 转化为资源奖励
- 降低某类 Debuff 的持续时间

### 2.2 不应提供能力

- 永久免疫 Debuff
- 无成本移除所有 Debuff
- 让风险奖励没有代价
- 自动选择最优 Debuff

### 2.3 示例联动

```text
结晶节点“修正棱镜”
→ 每 5 级修正窗口中，替换 Debuff 选项权重提升
→ 玩家更容易管理风险
→ 但仍不能指定移除最危险 Debuff
```

## 3. 与节点路线系统联动

水晶可以影响路线信息和节点奖励倾向。

### 3.1 路线预览

星核分支可提供：

- 预览后续节点类型
- 提前看到精英节点的大致奖励类型
- 显示风险节点的风险等级

### 3.2 特殊节点

高阶水晶可解锁：

- 元素试炼节点
- 水晶共鸣节点
- 分支专属事件
- 风险净化节点

这些节点不应过于频繁，每章最多出现 1 次。

## 4. 与角色系统联动

角色有元素倾向，水晶有分支倾向。

### 4.1 一致倾向

如果角色倾向与水晶投资方向一致：

- 相关奖励权重略微提升
- 角色专属任务更容易触发
- 羁绊资源获取增加

### 4.2 交叉倾向

如果角色倾向与水晶方向不同：

- 可形成混合 build
- 但成型更依赖局内选择

### 4.3 避免绑定死

不要让某个角色只能玩某个水晶分支。

---

## 5. 与武器系统联动

水晶不直接发放武器，但可以影响：

- 某类武器标签权重
- 某类状态附着奖励权重
- 武器升阶材料出现概率
- 商店中某类武器的刷新权重

示例：

```text
凝流投资较高
→ 法术近战 / 控制类武器在商店中略微更容易出现
→ 冻结相关状态升阶奖励权重提升
```

---

## 6. 与怪物系统联动

怪物可以掉落对应元素晶片。

示例：

| 怪物类型 | 可能掉落 |
|---|---|
| 电弧浮游 | 导能晶片 |
| 雾沫菇 | 生息晶片 |
| 霜镜守卫 | 凝流晶片 |
| 熔芯暴徒 | 灼焰晶片 |
| 晶壳甲虫 | 结晶晶片 |

Boss 可掉落古树芽核或分支关键材料。

---

## 7. 与商店系统联动

星核与结晶分支可增强商店策略：

- 首次刷新折扣
- 指定标签商品权重提升
- 可购买 Debuff 修正服务
- 可用元素晶片换取对应奖励

商店不应成为必经最优节点，价格和机会成本需要控制。
