# 怪物设计总览（Monsters Design）

> 版本：v1.0  
> 适用项目：《星链回响》（Echoes of Astra）  
> 当前目标：为 MVP / Early Build 提供一套可落地的怪物生态，用于验证元素状态、构筑流派、节点推进和战斗节奏。

---

## 1. 设计分布

本次设计共 **18 个怪物**：

| 类型 | 数量 | 设计目标 |
| --- | ---: | --- |
| 普通怪 | 8 | 提供基础怪潮、远程压制、区域干扰、辅助标记等战斗问题 |
| 精英怪 | 5 | 形成局部战斗目标，检验玩家 build、控制、爆发与优先级判断 |
| 特殊机制怪 | 2 | 改变战场规则，制造地图目标或阵地压力 |
| Boss | 3 | 综合检验阶段战、状态系统、清怪与风险管理 |

## 3. 投放建议

### 前期
- 星屑史莱姆
- 晶壳甲虫
- 针叶鼠

目标：建立基础怪潮、绕位、破甲和控制教学。

### 中期
- 火绒灯灵
- 雾沫菇
- 电弧浮游
- 标旗傀儡
- 熔芯暴徒
- 腐花园丁

目标：引入灼烧、中毒、感电、标记等状态联动，并开始要求玩家判断优先级。

### 后期
- 碎晶投手
- 霜镜守卫
- 黑晶收割者
- 裂隙孢巢
- 星轨哨塔

目标：制造高压场景，检验玩家 build 成型后的清怪、控制、爆发和生存能力。

### Boss 节点
- 星骸巨像：结晶护盾与破甲检验
- 雷鸣偶核：导能连锁与清怪优先级检验
- 腐耀花皇：生息毒雾、治疗花与灼烧清场检验

---

## 4. 后续扩展方向

1. 增加每个地图主题的专属怪物族群。
2. 为 Endless 模式设计变体怪，例如带词缀的强化普通怪。
3. 增加 Boss 专属状态反制机制，但避免完全否定玩家 build。
4. 为不同角色增加专属宿敌或剧情怪。
5. 增加怪物图鉴、掉落材料和水晶成长之间的联系。
