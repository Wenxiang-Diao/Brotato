# 寒镜匕首

## 📌 基本信息
- 武器名称：寒镜匕首
- 类型：法术 / 近战
- 定位：控制 / 爆发
- 初始品阶：蓝

## 🧱 基础属性
- 攻击力：中
- 攻速：快
- 范围：短
- 暴击率：高
- 击退：低
- 状态附着概率：中

## 🧪 状态附带
- 状态1：冻结｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：从侧面或背面对冻结目标命中。
- 触发效果：触发镜裂，对目标造成高暴击倍率伤害。
- 状态交互：冻结 + 爆发攻击形成破碎思路，但更偏单体刺杀。
- Build 联动：适合搭配冻结区域、减速和标记锁定。
- 限制条件：正面攻击不触发镜裂。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高侧背命中暴击率。
- 紫 → 粉：镜裂对标记目标额外增伤。
- 粉 → 红：镜裂后短时间提高移动速度。
- 红 → 金：镜裂击杀敌人时刷新一次短距离突进。

## 🧬 状态升阶
- 状态成长方式：冻结通过重复状态升阶。
- 叠加变化：冻结层数越高，侧背攻击窗口越长。
- 与状态系统的关系：控制类状态优先级高，但对 Boss 需要接受免疫或削弱。

## 🧠 构筑方向
- 适合流派：冻结破碎流 / 单体刺杀流。
- 推荐搭配状态：冻结、标记、破甲。
- 推荐搭配武器类型：区域冻结、精准标记、重击武器。
- 不适合搭配：纯自动清怪且不调整站位的构筑。

## ⚖️ 优缺点
- 优势：操作上限高，冻结后爆发强。
- 劣势：站位要求高，容错率低。

## 🧩 平衡性备注
- 强度来源：侧背冻结目标的镜裂暴击。
- 限制方式：方向条件、短范围和控制前置。
- 弱势场景：包围战、无法绕后的大型 Boss。
