# 赤晶祭刃

## 📌 基本信息
- 武器名称：赤晶祭刃
- 类型：法术 / 近战
- 定位：高风险 / 爆发
- 初始品阶：红

## 🧱 基础属性
- 攻击力：高
- 攻速：中
- 范围：短
- 暴击率：高
- 击退：中
- 状态附着概率：中

## 🧪 状态附带
- 状态1：标记｜紫
- 状态2：灼烧｜蓝
- 状态3：无

## 🔗 核心机制
- 触发条件：主动攻击前玩家生命高于最低保护线。
- 触发效果：消耗少量生命强化下一次斩击，对标记目标引燃并造成爆发。
- 状态交互：标记提高受伤，灼烧可接入中毒爆裂或过载前置。
- Build 联动：适合搭配吸血、护盾、低生命增益和爆发窗口。
- 限制条件：生命过低时不再消耗生命，也不触发强化斩击。

## 🔧 合成 / 升阶
- 蓝 → 紫：降低生命消耗。
- 紫 → 粉：强化斩击对标记目标伤害提高。
- 粉 → 红：引燃后短时间提升自身暴击率。
- 红 → 金：击杀标记目标时回复部分已消耗生命。

## 🧬 状态升阶
- 状态成长方式：标记优先升阶，灼烧通过重复状态补足。
- 叠加变化：标记层数提升爆发收益，灼烧层数提升后续反应空间。
- 与状态系统的关系：高收益绑定生命风险，符合 debuff 风险管理方向。

## 🧠 构筑方向
- 适合流派：标记爆发流 / 高风险收益流。
- 推荐搭配状态：标记、灼烧、吸血。
- 推荐搭配武器类型：续航武器、护盾辅助、Boss 单体武器。
- 不适合搭配：无回复能力的玻璃大炮构筑。

## ⚖️ 优缺点
- 优势：爆发上限高，适合处理精英和 Boss。
- 劣势：消耗生命，失误成本高。

## 🧩 平衡性备注
- 强度来源：生命换伤害与标记引燃。
- 限制方式：生命保护线、近战距离和回复需求。
- 弱势场景：持续消耗战、回复被压制的节点。
