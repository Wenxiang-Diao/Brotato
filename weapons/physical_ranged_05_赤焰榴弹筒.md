# 赤焰榴弹筒

## 📌 基本信息
- 武器名称：赤焰榴弹筒
- 类型：物理 / 远程
- 定位：爆发 / 清怪
- 初始品阶：红

## 🧱 基础属性
- 攻击力：高
- 攻速：慢
- 范围：远
- 暴击率：低
- 击退：高
- 状态附着概率：中

## 🧪 状态附带
- 状态1：灼烧｜紫
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：榴弹落地或命中敌人后短暂延迟。
- 触发效果：产生范围爆炸并附加灼烧，若目标已中毒则触发爆裂。
- 状态交互：灼烧 + 中毒形成爆裂范围伤害。
- Build 联动：适合搭配中毒扩散、吸附聚怪和范围增益。
- 限制条件：爆炸有延迟，快速敌人可能离开范围。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高爆炸半径。
- 紫 → 粉：对中毒目标爆裂伤害提高。
- 粉 → 红：榴弹中心区域附加更高灼烧概率。
- 红 → 金：爆裂击杀敌人时留下短暂燃烧区域。

## 🧬 状态升阶
- 状态成长方式：灼烧通过重复状态升阶。
- 叠加变化：灼烧层数提高 DOT 和爆裂前置稳定性。
- 与状态系统的关系：爆裂属于双状态交互，不能仅靠灼烧形成最高收益。

## 🧠 构筑方向
- 适合流派：灼烧中毒爆裂流 / 范围清怪流。
- 推荐搭配状态：灼烧、中毒、吸附。
- 推荐搭配武器类型：毒雾、聚怪、区域控制武器。
- 不适合搭配：纯单体标记构筑。

## ⚖️ 优缺点
- 优势：范围爆发强，清怪效率高。
- 劣势：射击节奏慢，命中和时机要求高。

## 🧩 平衡性备注
- 强度来源：范围爆炸和灼烧中毒交互。
- 限制方式：低攻速、延迟爆炸和状态前置。
- 弱势场景：高速精英、移动频繁的 Boss。
