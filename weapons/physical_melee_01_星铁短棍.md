# 星铁短棍

## 📌 基本信息
- 武器名称：星铁短棍
- 类型：物理 / 近战
- 定位：输出 / 控制
- 初始品阶：蓝

## 🧱 基础属性
- 攻击力：中
- 攻速：快
- 范围：短
- 暴击率：中
- 击退：中
- 状态附着概率：中

## 🧪 状态附带
- 状态1：破甲｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：连续命中同一目标 3 次。
- 触发效果：追加一次短距离重击，优先刷新目标身上的破甲层数。
- 状态交互：破甲目标受到高频攻击时更容易形成穿透收益。
- Build 联动：适合搭配高攻速、暴击和标记类武器，集中处理精英与 Boss。
- 限制条件：切换目标会重置连续命中计数。

## 🔧 合成 / 升阶
- 蓝 → 紫：提升基础攻击力，重击所需命中次数不变。
- 紫 → 粉：重击对破甲目标额外提高暴击率。
- 粉 → 红：重击可对近距离第二目标造成衰减伤害。
- 红 → 金：连续命中破甲目标时，短时间提高自身攻速。

## 🧬 状态升阶
- 状态成长方式：通过重复获得破甲状态并满足同状态同品阶数量条件升阶。
- 叠加变化：破甲层数提升后，重击收益更稳定。
- 与状态系统的关系：遵守单状态最多 3 层、单目标最多 3 种状态的限制。

## 🧠 构筑方向
- 适合流派：破甲高频流 / Boss 单体流。
- 推荐搭配状态：破甲、标记、感电。
- 推荐搭配武器类型：高攻速近战、精准远程、连锁触发武器。
- 不适合搭配：低攻速且频繁换目标的爆发武器。

## ⚖️ 优缺点
- 优势：机制直观，单体稳定，适合新手理解破甲价值。
- 劣势：范围小，面对大量分散敌人效率偏低。

## 🧩 平衡性备注
- 强度来源：连续命中后的重击与破甲叠层。
- 限制方式：需要保持贴身输出和目标锁定。
- 弱势场景：高机动敌人、密集弹幕、远程压力场景。
