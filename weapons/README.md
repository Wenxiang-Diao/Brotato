## 分类设计思路

- 物理近战：强调贴身风险、打击反馈和状态触发窗口，适合作为高频叠层、控制破绽与风险收益的入口。
- 物理远程：强调射程、弹道和目标选择，承担新手稳定输出、清怪弹射、蓄力控制和范围爆发。
- 法术近战：强调近距离法术效果与状态扩散，兼顾操作站位、状态传播和爆发触发。
- 法术远程：强调区域、自动化和组合反应，是状态联动、控场与后期构筑上限的主要来源。
