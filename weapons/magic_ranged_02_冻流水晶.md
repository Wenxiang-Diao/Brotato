# 冻流水晶

## 📌 基本信息
- 武器名称：冻流水晶
- 类型：法术 / 远程
- 定位：控制 / 辅助
- 初始品阶：蓝

## 🧱 基础属性
- 攻击力：低
- 攻速：慢
- 范围：远
- 暴击率：低
- 击退：低
- 状态附着概率：高

## 🧪 状态附带
- 状态1：冻结｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：在敌人密集区域生成冻流场。
- 触发效果：区域内敌人持续减速并周期性尝试附加冻结。
- 状态交互：冻结为破碎、镜裂和重击爆发提供前置。
- Build 联动：适合搭配板砖、冷凝弩、寒镜匕首等冻结收益武器。
- 限制条件：区域固定，敌人离开后收益停止。

## 🔧 合成 / 升阶
- 蓝 → 紫：冻流场持续时间增加。
- 紫 → 粉：区域内冻结附着概率提高。
- 粉 → 红：冻结目标受到下一次重击伤害提高。
- 红 → 金：冻流场结束时对仍处于冻结的敌人造成一次冰裂伤害。

## 🧬 状态升阶
- 状态成长方式：冻结通过重复状态升阶。
- 叠加变化：冻结层数提高控制稳定性。
- 与状态系统的关系：控制类优先级高，但对 Boss 需按免疫或削弱规则处理。

## 🧠 构筑方向
- 适合流派：控制区域流 / 冻结爆发前置。
- 推荐搭配状态：冻结、破甲、标记。
- 推荐搭配武器类型：重击近战、蓄力远程、侧背爆发武器。
- 不适合搭配：持续高速移动且不围绕区域作战的流派。

## ⚖️ 优缺点
- 优势：控场强，能明显降低生存压力。
- 劣势：伤害低，区域固定。

## 🧩 平衡性备注
- 强度来源：持续冻结和区域控场。
- 限制方式：低伤害、固定区域和控制抗性。
- 弱势场景：高机动敌人、Boss 免控阶段。
