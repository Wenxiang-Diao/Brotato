# 腐蚀喷瓶

## 📌 基本信息
- 武器名称：腐蚀喷瓶
- 类型：法术 / 远程
- 定位：状态 / 持续
- 初始品阶：紫

## 🧱 基础属性
- 攻击力：低
- 攻速：中
- 范围：中
- 暴击率：低
- 击退：低
- 状态附着概率：高

## 🧪 状态附带
- 状态1：中毒｜蓝
- 状态2：破甲｜蓝
- 状态3：无

## 🔗 核心机制
- 触发条件：向前方扇形区域喷洒腐蚀雾。
- 触发效果：区域内敌人持续受到中毒判定，并有概率附加破甲。
- 状态交互：中毒 + 破甲同时提高持续伤害和后续物理输出收益。
- Build 联动：适合搭配灼烧爆裂、破甲穿透和毒扩散。
- 限制条件：射程有限，持续喷洒时需要保持方向。

## 🔧 合成 / 升阶
- 蓝 → 紫：腐蚀雾范围增加。
- 紫 → 粉：破甲附着概率提高。
- 粉 → 红：满层中毒目标受到破甲附着概率额外提高。
- 红 → 金：腐蚀雾命中 5 个以上敌人时留下短暂残留区域。

## 🧬 状态升阶
- 状态成长方式：中毒和破甲均可通过重复状态升阶。
- 叠加变化：中毒提高持续输出，破甲提高后续打击收益。
- 与状态系统的关系：一个武器提供两种状态，但单目标总状态仍受 3 种限制。

## 🧠 构筑方向
- 适合流派：中毒破甲持续流 / 状态铺垫流。
- 推荐搭配状态：中毒、破甲、灼烧。
- 推荐搭配武器类型：榴弹、链锯、裂片弹弓、毒芽镰。
- 不适合搭配：长距离风筝且不愿接近敌群的构筑。

## ⚖️ 优缺点
- 优势：状态覆盖优秀，能同时服务 DOT 和物理输出。
- 劣势：射程中短，直接伤害低。

## 🧩 平衡性备注
- 强度来源：双状态覆盖和持续区域判定。
- 限制方式：低基础伤害、方向控制和射程限制。
- 弱势场景：高速贴脸敌人、远程消耗战。
