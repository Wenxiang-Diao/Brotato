# 裂纹板砖

## 📌 基本信息
- 武器名称：裂纹板砖
- 类型：物理 / 近战
- 定位：控制 / 爆发
- 初始品阶：蓝

## 🧱 基础属性
- 攻击力：高
- 攻速：慢
- 范围：短
- 暴击率：低
- 击退：高
- 状态附着概率：中

## 🧪 状态附带
- 状态1：冻结｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：命中已冻结目标。
- 触发效果：触发破碎，对目标和周围小范围敌人造成爆发伤害。
- 状态交互：冻结 + 重击形成破碎，是控制转爆发的基础组合。
- Build 联动：适合搭配区域冻结、蓄力控制或聚怪武器。
- 限制条件：未冻结目标只造成普通重击，不触发破碎。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高冻结附着概率。
- 紫 → 粉：破碎范围小幅增加。
- 粉 → 红：破碎可对次要目标附加低品阶冻结。
- 红 → 金：破碎命中 3 个以上目标时返还一次攻击冷却。

## 🧬 状态升阶
- 状态成长方式：冻结状态随重复状态合成升阶。
- 叠加变化：冻结层数越高，破碎触发窗口越稳定。
- 与状态系统的关系：Boss 可免疫或削弱控制，但仍可受到破碎伤害的平衡版本。

## 🧠 构筑方向
- 适合流派：冻结爆发流 / 控制破碎流。
- 推荐搭配状态：冻结、破甲、标记。
- 推荐搭配武器类型：区域控制、聚怪、慢速高伤武器。
- 不适合搭配：纯中毒持续流。

## ⚖️ 优缺点
- 优势：控制后爆发明确，打击反馈强。
- 劣势：攻速慢，严重依赖冻结前置。

## 🧩 平衡性备注
- 强度来源：冻结目标上的破碎爆发。
- 限制方式：低频攻击和前置状态需求。
- 弱势场景：控制抗性高的 Boss、快速分散的小怪群。
