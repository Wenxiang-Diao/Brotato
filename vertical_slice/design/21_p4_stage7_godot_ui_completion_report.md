# P4 阶段 7：Godot 正式 UI/UX 完成报告

## 1. 完成范围

- 将用户批准的 22 个图标导入 `assets/ui/icons/`。
- 建立 `CanvasLayer + Control + Container` UI 根结构。
- 使用正式 HUD 替换旧程序化 HUD。
- 重做主菜单、三步教程、奖励、修正、暂停、设置、确认和结算页面。
- 加入鼠标、键盘和手柄焦点导航及最近设备提示。
- 加入 UI 缩放和低动态模式。
- 增加 UI 专项运行测试，并完成全量回归。

## 2. 运行时结构

```text
Main (Node2D)
└── UIRoot (CanvasLayer)
    ├── HUD
    ├── ScreenStack
    ├── ModalStack
    ├── ToastLayer
    └── DebugLayer
```

世界、战斗实体、预警和命中特效继续由 `Main` 绘制。正式页面不再调用旧 `_draw_hud()`、`_draw_menu()` 或覆盖层绘制函数。

## 3. 输入与可访问性

- 键盘：方向键、Enter、Esc、R、Tab、P。
- 鼠标：悬停同步焦点，单击确认。
- 手柄：D-pad/系统 UI 方向、A/B/X/Y、Start，左摇杆移动，右扳机主动技能。
- 危险操作使用确认弹窗，默认焦点位于取消。
- 支持 90%、100%、110%、125% UI 缩放。
- 低动态模式关闭屏幕震动和命中停顿，并降低闪屏强度。
