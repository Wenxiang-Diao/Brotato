# 战斗与数值唯一规格

> 本文件是垂直切片战斗规则的唯一事实来源。代码和数据若与本文件冲突，应修改代码或数据。

## 1. 操作

| 行为 | 输入 | 规则 |
|---|---|---|
| 移动 | WASD / 方向键 | 八方向自由移动 |
| 普通攻击 | 自动 | 所有武器独立计时并自动寻找目标 |
| 主动技能 | Space | 露米释放持续 5 秒、跟随角色移动的星辉回路 |
| 选择奖励 | 数字键 1-3 | 游戏暂停期间三选一 |
| 调试跳层 | F10 | 仅用于开发验证 |

玩家没有手动瞄准、翻滚、冲刺和攻击按钮。
使用 F10 的局会在指标中标记为调试局，不进入正式 A/B 汇总。

## 2. 玩家基础值

| 属性 | 初始值 |
|---|---:|
| 最大生命 | 100 |
| 移动速度 | 245 px/s |
| 伤害倍率 | 1.0 |
| 攻速倍率 | 1.0 |
| 状态时长倍率 | 1.0 |
| 主动技能冷却 | 18 秒 |
| 受击无敌 | 0.55 秒 |
| 初始武器 | 弹壳手枪 |

## 3. 武器槽位

- 最多同时拥有 4 把固定武器。
- 武器不存在随机词条、随机状态和品阶。
- 每把武器最高 3 级。
- 新武器与武器升级均占用奖励选项。
- 已满级武器不再进入奖励池。
- 第 2 至第 4 级的奖励中至少提供 1 个尚未解锁的武器，确保切片能够测试组合构筑。
- 风险模式从第 3 级开始，每次升级最多展示 1 个风险奖励。
- 玩家最多同时持有 5 个 Debuff；达到上限后风险奖励暂时退出奖励池。

## 4. 目标选择

- 弹壳手枪和磁暴硬币攻击距离玩家最近的存活敌人。
- 弹壳手枪每第 5 发变为 1.8 倍伤害精准弹，优先攻击已标记目标，其次攻击当前生命最高目标。
- 冻流水晶选择局部敌人数量最多的敌人位置。
- 裂纹板砖攻击玩家周围固定半径内的所有敌人。
- 投射物不会自动穿透；磁暴硬币可弹射。

## 5. 伤害公式

```text
护甲减伤后伤害 =
基础伤害
× 玩家伤害倍率
× 反应倍率
× 100 / (100 + 敌人护甲)
```

处理顺序：

1. 读取武器基础伤害与武器等级倍率。
2. 应用玩家全局伤害倍率。
3. 检查状态反应并应用反应倍率。
4. 应用护甲。
5. 优先扣除护盾。
6. 扣除生命。
7. 生命小于等于 0 时结算死亡与经验。

## 6. 武器等级

| 武器 | 等级成长 |
|---|---|
| 弹壳手枪 | 每级伤害 +25%；每第 5 发精准弹 |
| 磁暴硬币 | 每级伤害 +25%，弹射 +1 |
| 冻流水晶 | 每级半径 +8 px，持续时间 +0.5 秒 |
| 裂纹板砖 | 每级伤害 +30% |

## 7. 敌人与层数

普通敌人生命：

```text
实际生命 = 基础生命 × (1 + (当前层数 - 1) × 0.22)
```

生成间隔：

```text
基础间隔 = max(0.35, 1.05 - 当前层数 × 0.09)
实际间隔 = 基础间隔 / 敌人生成倍率
```

## 8. 六层节奏

| 层 | 内容 | 目标 |
|---:|---|---|
| 1 | 星屑史莱姆 | 熟悉移动与自动攻击 |
| 2 | 加入甲虫、针叶鼠 | 处理耐久与高速目标 |
| 3 | 加入火绒灯灵 | 处理远程弹体 |
| 4 | 加入标旗傀儡 | 建立目标优先级 |
| 5 | 熔芯暴徒 | 检验控制与爆发 |
| 6 | 星骸巨像 | 检验完整 Build |

- 第 1 至第 4 层各 90 秒。
- 第 5 层至少持续 90 秒，并且必须击败熔芯暴徒才能进入下一层。
- 第 6 层在击败 Boss 后结束。
- 升级选择与修正窗口不计入战斗时间。
- 目标总时长为 8 至 12 分钟。

## 9. 失败与胜利

- 玩家生命小于等于 0：失败。
- 星骸巨像死亡：胜利。
- 失败和胜利界面均可返回模式选择。

## 10. 性能预算

垂直切片目标：

- 同屏普通敌人软上限：150。
- 同屏投射物软上限：100。
- 目标帧率：60 FPS。
- 程序化图形阶段不评估正式特效性能。
