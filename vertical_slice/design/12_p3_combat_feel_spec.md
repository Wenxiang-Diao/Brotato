# P3 战斗手感唯一规格

> 启动日期：2026-06-07

## 目标

在不依赖正式 Sprite、粒子贴图和音频资产的情况下，让灰盒战斗具备清晰的命中层级、武器身份、受击公平性和 Boss 阶段节奏。

## 参考原则

- Brotato：默认自动攻击和 20 至 90 秒波次要求反馈快速、连续但不过度打断移动。
- Halls of Torment：能力、特性和物品协同需要不同攻击拥有可辨识的视觉身份。
- Deep Rock Galactic: Survivor：自动射击下，玩家主要决策是走位，因此危险攻击必须提前预警。
- Vampire Survivors 类设计：高密度攻击时不能为每次小伤害使用强停顿，否则输入与画面会变得不连续。

## 反馈层级

| 事件 | 命中闪白 | 击退 | 屏幕震动 | 命中停顿 |
|---|---:|---:|---:|---:|
| 手枪 / 硬币 | 0.07 秒 | 低 | 无 | 无 |
| 冰区 / 灼烧 | 0.06 秒 | 无 | 无 | 无 |
| 板砖重击 | 0.11 秒 | 高 | 4 px | 0.035 秒 |
| 状态反应 | 0.12 秒 | 中 | 5-7 px | 0.025-0.045 秒 |
| 玩家受击 | 玩家闪红 0.18 秒 | 中 | 7 px | 无 |
| Boss 砸地 | 红色危险区 | 玩家击退 | 9 px | 0.04 秒 |

命中停顿同时最多 0.05 秒；连续效果只取较大值，不累加。

## 武器身份

- 弹壳手枪：短直线枪口闪光；第 5 发精准弹使用更粗的白黄色轨迹。
- 磁暴硬币：蓝紫色脉冲与弹射连线。
- 冻流水晶：施法位置先出现扩张冰环，再生成持续区域。
- 裂纹板砖：玩家周围先出现短促蓄力圈，再产生高击退冲击环。
- 星辉回路：释放时产生双层金色扩张环，持续区域保持跟随。

## 敌人与玩家受击

- 所有敌人字段统一包含 `hit_flash` 和 `knockback_velocity`。
- 普通敌人承受 100% 击退，精英承受 55%，Boss 承受 25%。
- 玩家受击后维持原 0.55 秒无敌，并获得远离伤害来源的短促击退。
- 暂停恢复后提供 0.35 秒输入保护和无敌，避免立即受伤。

## Boss 阶段

星骸巨像保留初始护盾并增加三个可读阶段：

1. 护盾阶段：4.0 秒一次 1.1 秒砸地预警。
2. 核心暴露：护盾破裂后短暂停顿、碎晶反馈，攻击间隔缩短到 3.3 秒。
3. 残血阶段：生命低于 35% 后攻击间隔 2.6 秒，危险圈扩大，但预警时间不低于 0.9 秒。

阶段转换只发生一次，并通过文字、颜色和冲击环提示。Boss 不获得不可读的瞬发伤害。

## 性能约束

- 同时存在的程序化反馈效果上限为 80。
- 低价值 DOT 不生成屏幕震动。
- 连锁与范围伤害不会为每个目标分别触发命中停顿。
- 反馈使用独立时间与确定性振动函数，不消耗刷怪、奖励或掉落随机流。
