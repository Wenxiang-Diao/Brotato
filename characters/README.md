## 📌 角色系统设计目标

《星链回响》的角色系统强调“每角色定义玩法，武器为辅助，不是核心”。角色应通过被动技能、主动技能、终极技能、元素倾向、武器偏好和 debuff 处理方式，引导玩家形成不同的局内构筑路线。

## ✅ 设计分布

- **新手体验**：露米
- **状态连锁展示**：岚音
- **debuff 风险收益展示**：烬牙
- **控制与生存展示**：澪珀

这 4 名角色覆盖了 MVP 阶段最重要的玩法方向，可以用于验证：

- 元素反应是否足够清晰
- 状态叠加与交互是否有构筑深度
- debuff 风险管理是否有吸引力
- 水晶系统是否能支持角色长期成长
- 武器系统是否能作为角色玩法的辅助表达
