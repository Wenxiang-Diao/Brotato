# 《星链回响》完整策划归档

本目录保存项目当前阶段的完整策划设计。内容保持原有模块划分，用作长期版本设计来源。

## 目录

- `docs/`：总策划、元素系统、武器系统。
- `characters/`：角色框架与 4 名角色。
- `weapons/`：20 把武器及武器总览。
- `monsters_design_md/`：普通怪、精英、特殊怪与 Boss。
- `nodes/`：节点、路线、章节与节奏。
- `rewards/`：奖励池、品质、商店与经济。
- `debuffs/`：风险管理、修正窗口及 Debuff 示例。
- `crystals/`：水晶树与局外成长。
- `art_standard/`：美术、工程、数据和目录规范。
- `prompts/`：各模块设计提示词。

## 使用约定

- 本目录代表“完整版本策划”，不等于当前开发范围。
- 根目录的 `vertical_slice/` 是当前可执行垂直切片。
- 垂直切片验证通过后，再从本目录逐项吸收内容。
