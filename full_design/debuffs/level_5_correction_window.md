# 每 5 级修正窗口设计

---

## 1. 系统定位

每 5 级修正窗口是 Debuff 系统的核心安全阀。

它让玩家在不断接受风险的同时，拥有周期性整理构筑的机会，避免 Debuff 积累导致体验失控。

```text
等级提升
→ 累积奖励与 Debuff
→ 到达 5 / 10 / 15 / 20 ... 级
→ 进入修正窗口
→ 移除、压制、替换或转化 Debuff
→ 继续构筑
```

---

## 2. 触发规则

| 条件 | 规则 |
|---|---|
| 触发等级 | 每 5 级触发一次 |
| 触发时机 | 升级奖励选择完成后立即触发 |
| 是否可跳过 | 可以跳过，但会获得少量补偿资源 |
| 是否进入暂停 | 进入暂停选择界面 |
| 是否影响节点 | 不消耗节点行动 |

---

## 3. 基础选项池

修正窗口提供 3 个选项，类似抽奖三选一。

基础选项包括：

| 选项 | 效果 |
|---|---|
| 移除 | 移除 1 个轻度 Debuff |
| 降层 | 指定 1 个 Debuff 层数 -1 |
| 压制 | 指定 1 个 Debuff 在下一节点失效 |
| 替换 | 将 1 个 Debuff 替换为同等级随机 Debuff |
| 转化 | 将 1 个 Debuff 的部分代价转为收益 |
| 净化 | 移除 1 个绑定 Debuff，但对应奖励降级 |
| 延迟 | 当前 Debuff 不变，获得金币 / 水晶碎片补偿 |

---

## 4. 选项生成规则

推荐规则：

1. 如果玩家没有 Debuff，修正窗口转为“强化窗口”。
2. 如果玩家有 1-2 个 Debuff，提供 1 个移除选项、1 个强化选项、1 个替换选项。
3. 如果玩家有 3-4 个 Debuff，至少提供 2 个风险管理选项。
4. 如果玩家有 5 个 Debuff，必定提供 1 个移除或降层选项。
5. 如果玩家持有诅咒，提供低概率诅咒压制或代价交换选项。

---

## 5. 选项品质

修正窗口的选项也可以有品质，但不建议过度复杂。

| 品质 | 示例 |
|---|---|
| 蓝 | 压制 1 个轻度 Debuff 1 节点 |
| 紫 | 移除 1 个轻度 Debuff |
| 粉 | 降低 1 个中度 Debuff 层数 |
| 红 | 移除 1 个中度 Debuff 或压制重度 Debuff |
| 金 | 将 1 个 Debuff 转化为特殊收益，但保留部分代价 |

---

## 6. 与角色联动

| 角色方向 | 修正窗口特色 |
|---|---|
| 新手均衡型 | 第一次修正窗口额外 +1 选项 |
| 状态连锁型 | 状态类 Debuff 可转化为状态反应加成 |
| 风险收益型 | 可以选择保留 Debuff 并获得临时强化 |
| 控制 / 生存型 | 更容易压制操作类和环境类 Debuff |

---

## 7. 与水晶系统联动

水晶树可以提供：

| 水晶方向 | 效果 |
|---|---|
| 导能 | 状态类 Debuff 转化概率提高 |
| 生息 | 治疗 / 生命类 Debuff 修正效果提高 |
| 凝流 | 控制 / 移动类 Debuff 压制时长提高 |
| 灼焰 | 风险保留时获得额外伤害 |
| 结晶 | 防御 / 护盾类 Debuff 降层效果提高 |
| 通用 | 修正窗口选项 +1 或刷新成本降低 |

---

## 8. 与节点系统联动

修正窗口不替代休息节点。

两者区别：

| 系统 | 作用 |
|---|---|
| 修正窗口 | 周期性整理，保证局内安全阀 |
| 休息节点 | 玩家主动选择的恢复 / 净化节点 |
| 商店节点 | 花费资源管理风险 |
| 事件节点 | 高波动风险交换 |

---

## 9. 跳过修正

玩家可以跳过修正窗口，获得补偿。

补偿建议：

- 金币 +15
- 当前 build 标签权重 +5%
- 下一次抽奖免费刷新 1 次
- 水晶碎片 +1

风险型角色或水晶可强化跳过收益。

---

## 10. UI 表现

修正窗口界面需要显示：

- 当前 Debuff 列表
- Debuff 危险等级
- Debuff 层数
- Debuff 来源
- 是否绑定奖励
- 预计修正结果
- 跳过补偿

玩家必须能在选择前理解后果。

---

## 11. MVP 推荐方案

MVP 阶段推荐先实现：

- 每 5 级触发
- 3 个修正选项
- 移除轻度 Debuff
- 降低 Debuff 层数
- 压制 Debuff 1 个节点
- 跳过获得金币
- Debuff 上限时强制提供修正选项

暂缓实现：

- 复杂转化
- 诅咒净化分支
- 多角色专属修正窗口动画
