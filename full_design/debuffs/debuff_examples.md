- 分类：属性类
- 危险等级：轻度
- 效果：移动速度 -6%，近战命中后 1 秒内恢复一半惩罚
- 推荐来源：升级奖励、风险节点、事件节点、强力奖励绑定
- 适合绑定奖励：根据分类绑定对应攻击、状态、资源或节点奖励
- 修正方式：修正窗口降层 / 休息节点压制 / 商店净化 / 特定水晶转化
- 风险流派可用：是
- 平衡性备注：该 Debuff 需要在 UI 中明确展示来源和层数，避免与同类重度 Debuff 过度叠加。

