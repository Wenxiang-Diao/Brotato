# ⚠️ 风险收益与 Debuff 绑定规则

> 目标：让“强力成长必然伴随风险”成为项目差异化核心，而不是惩罚玩家的随机负担。

---

## 1. 风险收益设计原则

| 原则 | 说明 |
|---|---|
| 代价可预期 | 玩家选择前必须看到 debuff 或限制 |
| 收益要明确 | 风险奖励必须明显强于普通奖励 |
| 可修正 | 玩家应能通过每 5 级修正窗口、休息、商店降低风险 |
| 可构筑 | 部分角色和水晶可以把 debuff 转化为收益 |
| 不强迫 | 主线构筑不能必须依赖高风险奖励才能通关 |

---

## 2. Debuff 绑定等级

| 绑定等级 | 适用奖励 | 示例代价 |
|---|---|---|
| 轻度 | 粉色、低风险红色 | 移速 -3%、拾取范围 -10%、下一节点敌人 +5% |
| 中度 | 红色、部分金色 | 最大生命 -8%、商店价格 +10%、状态抗性下降 |
| 重度 | 金色、特殊事件 | 每节点开始扣血、Boss 强化、无法移除某 debuff |
| 条件型 | 机制奖励 | 低血才生效、需要 3 状态反应、触发后冷却 |

---

## 3. 风险奖励类型

### 3.1 直接代价型

获得奖励时立即附加 debuff。

示例：

```text
赤焰终幕：灼烧引爆效果大幅增强，但获得“热浪疲劳”：技能冷却 +8%。
```

---

### 3.2 条件代价型

只有触发强效果时才产生代价。

```text
过载燃心：每次触发过载时伤害 +60%，但自身获得 1 层灼烧压力。
```

---

### 3.3 延迟代价型

奖励立即生效，但代价在之后节点出现。

```text
古树透支：立即获得 2 个粉色奖励，但接下来 2 个战斗节点敌人数量 +20%。
```

---

### 3.4 转化收益型

debuff 不再只是负面，而成为构筑资源。

```text
逆枝共鸣：每拥有 1 个 debuff，状态伤害 +6%，但 debuff 无法在休息节点完全移除。
```

---

## 4. Debuff 与奖励品质对应

| 品质 | Debuff 绑定建议 |
|---|---|
| 蓝 | 不绑定 |
| 紫 | 不绑定或事件中轻度绑定 |
| 粉 | 小概率绑定轻度 debuff |
| 红 | 建议绑定轻度 / 中度限制 |
| 金 | 必须绑定代价、条件或稀缺资源 |

---

## 5. Debuff 修正窗口

每 5 级提供一次修正窗口，玩家可选择：

| 选项 | 效果 |
|---|---|
| 净化 | 移除 1 个轻度 debuff |
| 替换 | 将 1 个中度 debuff 替换为随机轻度 debuff |
| 压缩 | 将 2 个轻度 debuff 合成 1 个中度 debuff，并获得奖励 |
| 接纳 | 保留 debuff，获得对应标签奖励 |
| 延迟 | 使 1 个 debuff 下一节点不生效 |

设计目的：让风险不是不可逆惩罚，而是持续管理过程。

---

## 6. 风险奖励出现条件

风险奖励不会在所有场景中出现。

| 场景 | 风险奖励出现率 |
|---|---:|
| 普通升级 | 低 |
| 普通战斗节点 | 低 |
| 精英节点 | 中高 |
| 事件节点 | 高 |
| 风险节点 | 很高 |
| Boss 后奖励 | 中 |
| 商店隐藏栏 | 中 |

---

## 7. 风险 UI 表达

风险奖励必须明确展示：

1. 奖励本体效果
2. 绑定 debuff 名称
3. debuff 严重程度
4. 是否可移除
5. 是否能被角色 / 水晶转化
6. 推荐 / 不推荐场景

示例展示：

```text
【红】过载燃心
效果：过载伤害 +60%。
代价：每触发 3 次过载，获得 1 层“热浪疲劳”。
风险等级：中
可修正：是
推荐：灼焰 / 风险流
```

---

## 8. 防止风险系统失控的规则

1. 同一局最多同时存在 5 个普通 debuff。
2. 重度 debuff 最多同时存在 1 个。
3. 如果玩家生命低于 25%，系统降低重度风险奖励权重。
4. 新手角色默认降低风险奖励出现率。
5. 风险型角色可以提高风险收益，但不能免疫代价。
6. Boss 前一层至少提供 1 个修正机会或恢复机会。

---

## 9. 风险构筑示例

### 9.1 低血爆发流

```text
低生命 → 攻击提高 → 获得生命上限降低 debuff → 通过吸血维持生存
```

### 9.2 Debuff 转状态流

```text
每个 debuff 提高状态伤害 → 选择中毒 / 灼烧 / 感电持续输出 → 通过修正窗口控制 debuff 数量
```

### 9.3 高压清怪流

```text
敌人数量增加 → 击杀收益提高 → AOE 和连锁奖励收益提高 → 风险节点收益最大化
```
