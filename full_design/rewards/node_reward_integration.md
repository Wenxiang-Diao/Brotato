# 🗺️ 节点奖励联动规则

> 目标：让节点路线选择与奖励系统强绑定，使玩家根据当前 Build、风险、资源和血量选择不同路线。

---

## 1. 节点奖励总览

| 节点类型 | 奖励强度 | 风险 | 主要奖励 |
|---|---:|---:|---|
| 普通战斗 | 中低 | 低 | 经验、金币、低阶奖励 |
| 精英 | 高 | 高 | 高品质奖励、武器升阶、风险奖励 |
| 事件 | 波动 | 中高 | 特殊奖励、代价交换 |
| 商店 | 可控 | 低 | 购买、刷新、修正 |
| 抽奖 | 中高 | 低中 | 免费三选一 |
| 休息 | 中低 | 低 | 回复、净化、防御 Buff |
| 宝藏 / 水晶 | 中 | 低 | 资源、晶尘、局外成长 |
| 风险节点 | 很高 | 很高 | 高品质奖励 + debuff |
| Boss | 很高 | 高 | 红 / 金奖励、晶尘、解锁 |

---

## 2. 普通战斗节点

### 奖励结构

```text
固定：经验 + 回响币
概率：蓝 / 紫奖励三选一
低概率：粉色奖励
```

### 设计目的

- 提供稳定成长。
- 让玩家在安全路线中也能推进 Build。
- 不产生过强奖励，避免普通节点价值超过精英节点。

---

## 3. 精英节点

### 奖励结构

```text
固定：较多经验 + 回响币
必定：粉色或以上三选一
概率：红色 / 金色风险奖励
额外：武器升阶材料或状态核心
```

### 特殊规则

- 精英奖励至少 1 个与当前 Build 标签相关。
- 精英节点更容易出现风险奖励。
- 如果玩家血量低于 30%，风险奖励权重降低。

---

## 4. 事件节点

事件节点奖励不稳定，重点是选择。

### 标准结构

```text
安全选项：低收益，无代价
构筑选项：中收益，轻度代价
贪婪选项：高收益，明显代价
```

### 示例

| 事件 | 安全 | 构筑 | 贪婪 |
|---|---|---|---|
| 古树裂枝 | 获得 50 回响币 | 获得状态核心，失去 5% 生命 | 获得红色奖励，附加 debuff |
| 星尘赌桌 | 获得蓝色奖励 | 重抽 3 次取 1 | 获得随机粉 / 红奖励，但下节点敌人强化 |
| 失控晶炉 | 恢复生命 | 强化灼烧 / 感电 | 获得金色风险奖励，附加重度 debuff |

---

## 5. 商店节点

商店节点不直接给免费奖励，而是提供选择权。

### 商店特色

- 商品可见
- 可刷新
- 可锁定
- 可购买修正
- 可购买风险折扣商品

### 与路线关系

商店适合：

1. 当前金币充足
2. Build 缺关键件
3. debuff 需要修正
4. Boss 前补给
5. 高风险路线后的恢复

---

## 6. 抽奖节点

抽奖节点是免费构筑强化点。

### 奖励结构

```text
免费三选一
至少 1 个粉色或以上
较高概率出现当前 Build 标签奖励
可花费回响币重抽
```

### 设计目的

- 强化路线选择的期待感。
- 为运气较差的玩家提供构筑补偿。
- 作为章节中段的重要节奏点。

---

## 7. 休息节点

休息节点偏防御与修正。

### 奖励结构

玩家选择 1 项：

| 选项 | 效果 |
|---|---|
| 恢复 | 回复 35% 最大生命 |
| 净化 | 移除 1 个轻度 debuff |
| 稳固 | 获得防御 / 护盾类 Buff |
| 整备 | 免费刷新下一次商店 1 次 |
| 冥想 | 获得少量晶尘或角色羁绊点 |

---

## 8. 宝藏 / 水晶节点

### 奖励方向

- 回响币
- 晶尘
- 稀有资源
- 水晶临时祝福
- 低风险构筑补强

该节点风险低，但战斗成长不如精英或抽奖节点。

---

## 9. 风险节点

风险节点是高收益路线。

### 奖励结构

```text
进入前展示代价
完成后获得高品质奖励
至少 1 个红色选项
小概率金色
必定附加或强化 debuff
```

### 设计目的

- 服务风险型角色。
- 给高熟练度玩家提供挑战。
- 让路线选择有明显风格差异。

---

## 10. Boss 节点

Boss 节点是章节终点。

### 奖励结构

| 奖励 | 说明 |
|---|---|
| 高品质三选一 | 粉 / 红 / 金 |
| 晶尘 | 局外水晶升级 |
| Boss 核心 | 解锁高级内容 |
| 角色羁绊 | 使用角色通关后获得 |
| 图鉴解锁 | Boss 信息与机制记录 |

### Boss 奖励限制

- 金色奖励必须有代价。
- Boss 专属奖励应偏机制，而非纯数值。
- Boss 后允许一次免费 debuff 修正或水晶选择。

---

## 11. 章节奖励节奏

| 章节 | 奖励倾向 |
|---|---|
| 第 1 章 | 蓝 / 紫为主，帮助建立基础 Build |
| 第 2 章 | 粉色增多，开始形成核心联动 |
| 第 3 章 | 红色和风险奖励增多，要求玩家完成取舍 |
| Boss 后 | 提供阶段性高价值奖励和局外资源 |

---

## 12. 节点奖励平衡原则

1. 普通节点提供稳定收益。
2. 精英节点提供强奖励但需要承担战斗风险。
3. 事件节点提供不确定性和代价交换。
4. 商店提供可控性但消耗资源。
5. 休息节点降低死亡风险，但构筑收益低。
6. 风险节点奖励最高，但必须附带清晰代价。
7. Boss 节点承担阶段性成就与局外成长。
