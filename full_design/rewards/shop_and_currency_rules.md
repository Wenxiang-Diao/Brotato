# 🛒 商店与货币规则

> 目标：让商店成为玩家主动规划 Build、修正风险和消耗资源的核心节点。

---

## 1. 局内货币：回响币

回响币是局内主要货币。

### 获取方式

| 来源 | 数量范围 |
|---|---:|
| 普通怪掉落 | 少量 |
| 精英怪 | 中量 |
| 宝藏 / 水晶节点 | 中高 |
| 事件奖励 | 波动较大 |
| Boss | 高 |
| 放弃奖励转化 | 根据品质转化 |

### 用途

| 用途 | 说明 |
|---|---|
| 商店购买 | 购买武器、Buff、修正、资源 |
| 重抽 | 刷新三选一奖励 |
| 锁定商品 | 保留商店商品到下次商店 |
| Debuff 修正 | 净化、替换、延迟 |
| 特殊事件 | 支付代价换取奖励 |

---

## 2. 局外货币：晶尘

晶尘用于水晶系统升级。

| 来源 | 说明 |
|---|---|
| Boss 击败 | 主要来源 |
| 章节通关 | 稳定来源 |
| 高风险节点 | 额外来源 |
| 成就 / 图鉴 | 一次性奖励 |

晶尘不应大量出现在普通商店中，避免局外成长被刷币行为破坏。

---

## 3. 商店基础结构

商店默认 6 个商品槽。

| 槽位 | 商品类型 | 出现概率方向 |
|---|---|---|
| 1 | 武器 / 武器升阶 | 高 |
| 2 | 武器 / 状态核心 | 高 |
| 3 | Buff 模块 | 高 |
| 4 | Buff / 资源 | 中 |
| 5 | 回复 / Debuff 修正 | 固定概率 |
| 6 | 折扣 / 风险商品 / 特殊商品 | 波动 |

---

## 4. 商店价格规则

| 品质 | 基础价格 |
|---|---:|
| 蓝 | 35 |
| 紫 | 70 |
| 粉 | 130 |
| 红 | 240 |
| 金 | 400+ |

价格修正：

| 条件 | 修正 |
|---|---:|
| 当前章节 +1 | 价格 +15% |
| 商品与当前 Build 高匹配 | 价格 +10% |
| 风险商品 | 价格 -20% 到 -40% |
| 折扣商品 | 价格 -30% |
| 商店价格 debuff | 价格 +10% 到 +25% |

---

## 5. 刷新与锁定

### 5.1 刷新

玩家可以花费回响币刷新商店全部商品。

| 刷新次数 | 花费 |
|---|---:|
| 第 1 次 | 15 |
| 第 2 次 | 30 |
| 第 3 次 | 50 |
| 后续 | 每次 +25 |

### 5.2 锁定

玩家可以锁定最多 2 个商品。

| 锁定数量 | 花费 |
|---|---:|
| 1 个 | 10 |
| 2 个 | 25 |

锁定商品会在下一次商店继续出现。

---

## 6. 商品类型

### 6.1 武器商品

- 新武器
- 相同武器升阶材料
- 武器状态附魔
- 武器触发机制强化

### 6.2 状态核心商品

- 灼烧核心
- 中毒核心
- 冻结核心
- 感电核心
- 破甲核心
- 标记核心

### 6.3 Buff 商品

- 攻击 / 防御 / 频率 / 范围 / 回复 / 状态强化

### 6.4 修正商品

- 移除轻度 debuff
- 替换中度 debuff
- 延迟 debuff
- 回复生命
- 增加下一节点奖励

### 6.5 风险商品

风险商品通常价格较低，但附带代价。

示例：

```text
折扣红色奖励：价格 -40%，但购买后获得 1 个随机轻度 debuff。
```

---

## 7. 商店与节点路线关系

商店节点的价值取决于玩家路线选择。

| 玩家状态 | 商店价值 |
|---|---|
| 金币较多 | 可快速补强核心 Build |
| debuff 较多 | 可修正风险 |
| 血量较低 | 可购买恢复 |
| Build 缺关键件 | 可通过刷新寻找状态 / 武器 |
| 准备打 Boss | 商店成为安全补给点 |

---

## 8. 商店平衡原则

1. 商店不应保证刷到核心金色奖励。
2. 商店能提高构筑可控性，但需要资源成本。
3. 风险商品必须清楚展示代价。
4. 回复与 debuff 修正不能过于廉价，否则风险系统失效。
5. 高级商品出现率随章节提高，但价格也同步提高。

## 9. MVP 商店实现建议

MVP 只需实现：

1. 6 个商品槽
2. 基础品质与价格
3. 刷新
4. 锁定 1 个商品
5. 回复商品
6. 移除轻度 debuff 商品
7. 武器与 Buff 商品

后续再加入：

- 隐藏风险商人
- 角色专属商品
- 水晶专属折扣
- Boss 后黑市
