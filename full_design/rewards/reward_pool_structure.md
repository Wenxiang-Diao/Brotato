# 🧩 奖励池结构设计

> 目标：通过分层奖励池和标签权重，让玩家既能形成构筑方向，又保留 Roguelite 随机性。

---

## 1. 奖励池总结构

```text
总奖励池
├── 通用成长池
├── 武器池
├── 状态核心池
├── Buff 模块池
├── Debuff 修正池
├── 资源池
├── 角色强化池
├── 节点专属池
├── 风险奖励池
└── Boss / 稀有池
```

---

## 2. 通用成长池

适用于升级、普通战斗、低风险事件。

| 奖励类型 | 示例 | 作用 |
|---|---|---|
| 属性 Buff | 攻击力 +8%、攻速 +6% | 基础成长 |
| 生存 Buff | 最大生命 +10、拾取回血 | 容错提升 |
| 资源奖励 | 金币、经验球、重抽券 | 经济补充 |
| 低阶状态强化 | 灼烧时间 +0.5 秒 | 轻度 Build 引导 |

---

## 3. 武器池

用于提供新武器、已有武器升阶、武器机制强化。

| 子池 | 出现场景 | 说明 |
|---|---|---|
| 新武器池 | 升级、抽奖节点、商店 | 提供新构筑入口 |
| 升阶材料池 | 商店、精英、Boss | 提高已有武器品阶 |
| 武器特化池 | 高等级后 | 让武器获得额外状态或触发机制 |

武器池必须与四类武器保持一致：

- 物理近战
- 物理远程
- 法术近战
- 法术远程

---

## 4. 状态核心池

状态核心是构筑系统的重要组件。

| 状态核心 | 效果方向 |
|---|---|
| 灼烧核心 | 持续伤害、爆裂、火焰区域 |
| 中毒核心 | 叠层、扩散、长线伤害 |
| 冻结核心 | 控制、破碎、减速 |
| 感电核心 | 连锁、传导、高频触发 |
| 破甲核心 | Boss 单体、物理穿透 |
| 标记核心 | 集火、暴击、连锁目标 |

状态核心不应直接取代武器，而是强化“施加状态 → 状态交互 → 触发效果”的路径。

---

## 5. Buff 模块池

Buff 模块按功能分为：

| 类型 | 示例 | 适合流派 |
|---|---|---|
| 输出 | 攻击力、暴击、爆伤 | 爆发 / Boss |
| 频率 | 攻速、冷却缩减、多段触发 | 高频 / 连锁 |
| 范围 | 投射物范围、爆炸半径 | 清怪 / AOE |
| 生存 | 最大生命、护盾、吸血 | 生存 / 风险 |
| 状态 | 状态概率、状态层数、状态时长 | 状态流 |
| 资源 | 金币、经验、拾取范围 | 经济流 |
| 特殊 | 低血增伤、debuff 转收益 | 风险流 |

---

## 6. Debuff 修正池

用于降低高风险构筑失控的概率。

| 奖励 | 效果 |
|---|---|
| 净化叶片 | 移除 1 个轻度 debuff |
| 重塑晶片 | 将 1 个 debuff 替换为随机轻度 debuff |
| 压缩印记 | 将 2 个轻度 debuff 合并为 1 个中度 debuff，并获得额外奖励 |
| 暂缓符文 | 使 1 个 debuff 下一节点不生效 |
| 反转契约 | 保留 debuff，但获得对应收益 |

---

## 7. 资源池

| 资源 | 局内 / 局外 | 用途 |
|---|---|---|
| 回响币 | 局内 | 商店购买、重抽、修正 |
| 星屑经验 | 局内 | 升级与抽奖触发 |
| 晶尘 | 局外 | 水晶升级 |
| 羁绊残片 | 局外 | 角色羁绊成长 |
| Boss 核心 | 局外 / 解锁 | 高级水晶、武器、角色解锁 |

---

## 8. 节点专属池

| 节点 | 专属奖励方向 |
|---|---|
| 普通战斗 | 通用成长、低阶武器、金币 |
| 精英 | 高品质 Buff、武器升阶、风险奖励 |
| 商店 | 购买、刷新、锁定、修正 |
| 抽奖 | 免费高权重三选一 |
| 休息 | 恢复、净化、防御型 Buff |
| 事件 | 特殊奖励、代价交换、随机挑战 |
| Boss | 红 / 金奖励、水晶碎片、解锁材料 |

---

## 9. 风险奖励池

风险奖励池只在以下情况下出现：

1. 精英节点
2. 事件节点
3. Boss 后奖励
4. 玩家已有 2 个以上 debuff 时的特殊转化
5. 风险型角色或风险水晶激活后

风险池奖励强度高，但必须绑定代价。

---

## 10. 奖励池抽取流程

```text
确定触发来源
→ 确定基础奖励池
→ 应用品质概率
→ 应用角色倾向权重
→ 应用水晶权重
→ 应用当前 Build 标签权重
→ 检查保底与重复限制
→ 生成 3 个候选奖励
→ 玩家选择 1 个
```
