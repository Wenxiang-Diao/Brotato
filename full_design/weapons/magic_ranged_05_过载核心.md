# 过载核心

## 📌 基本信息
- 武器名称：过载核心
- 类型：法术 / 远程
- 定位：爆发 / 高风险
- 初始品阶：金

## 🧱 基础属性
- 攻击力：高
- 攻速：慢
- 范围：远
- 暴击率：中
- 击退：中
- 状态附着概率：中

## 🧪 状态附带
- 状态1：灼烧｜紫
- 状态2：感电｜紫
- 状态3：无

## 🔗 核心机制
- 触发条件：同一目标同时存在灼烧与感电。
- 触发效果：目标积累过载值，满值后发生范围爆炸。
- 状态交互：灼烧 + 感电转化为过载爆发，属于进阶构筑结果。
- Build 联动：适合搭配高频灼烧、感电连锁、标记集火和聚怪。
- 限制条件：启动慢，缺少任一状态时只表现为普通法术远程。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高灼烧与感电附着概率。
- 紫 → 粉：过载积累速度提高。
- 粉 → 红：过载爆炸范围增加。
- 红 → 金：过载爆炸命中标记目标时追加一次弱传导。

## 🧬 状态升阶
- 状态成长方式：灼烧与感电均可通过重复状态升阶。
- 叠加变化：两种状态层数越稳定，过载触发越可靠。
- 与状态系统的关系：该武器强依赖双状态组合，符合机制组合大于数值强化的原则。

## 🧠 构筑方向
- 适合流派：过载爆发流 / 高风险高收益流。
- 推荐搭配状态：灼烧、感电、标记、吸附。
- 推荐搭配武器类型：星焰腕刃、磁暴硬币、雷鸣星盘、聚怪辅助。
- 不适合搭配：缺少灼烧或感电覆盖的单状态构筑。

## ⚖️ 优缺点
- 优势：后期爆发上限高，适合清理高密度敌群。
- 劣势：启动慢，构筑要求高，早期单独使用弱。

## 🧩 平衡性备注
- 强度来源：双状态共存后的过载爆炸。
- 限制方式：高初始品阶、慢攻速、状态前置和积累时间。
- 弱势场景：短战斗节点、状态抗性敌人、单体移动 Boss。
