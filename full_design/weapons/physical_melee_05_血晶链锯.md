# 血晶链锯

## 📌 基本信息
- 武器名称：血晶链锯
- 类型：物理 / 近战
- 定位：持续 / 高风险 / 续航
- 初始品阶：红

## 🧱 基础属性
- 攻击力：中
- 攻速：极快
- 范围：短
- 暴击率：低
- 击退：低
- 状态附着概率：高

## 🧪 状态附带
- 状态1：中毒｜紫
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：持续贴近同一目标进行切割。
- 触发效果：快速叠加中毒，玩家生命低于 40% 时，对满层中毒目标造成伤害可回复少量生命。
- 状态交互：中毒可与灼烧触发爆裂，也可通过扩散形成持续流。
- Build 联动：适合搭配防御削弱、回血增益和风险型 debuff。
- 限制条件：离开目标会快速丢失切割节奏加成。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高中毒附着频率。
- 紫 → 粉：低生命吸血阈值提高到 50%。
- 粉 → 红：对满层中毒目标提升切割伤害。
- 红 → 金：生命低于 30% 时，满层中毒目标死亡会产生小范围毒雾。

## 🧬 状态升阶
- 状态成长方式：中毒通过重复状态升阶；回复效果属于武器机制，不作为附带状态参与升阶。
- 叠加变化：中毒层数越高，持续伤害和低生命回复触发越稳定。
- 与状态系统的关系：遵守单状态 3 层上限，不能通过超层数无限放大回复。

## 🧠 构筑方向
- 适合流派：中毒续航流 / 高风险持续流。
- 推荐搭配状态：中毒、破甲、扩散。
- 推荐搭配武器类型：防御辅助、聚怪、持续区域武器。
- 不适合搭配：远程风筝流和低风险保守构筑。

## ⚖️ 优缺点
- 优势：持续输出强，低生命时具备反打能力。
- 劣势：需要贴身，操作失败时死亡风险高。

## 🧩 平衡性备注
- 强度来源：高频叠毒和低生命回复窗口。
- 限制方式：短范围、低击退和生命阈值风险。
- 弱势场景：爆发伤害敌人、控制免疫 Boss、远程压制场景。
