# 旋刃手斧

## 📌 基本信息
- 武器名称：旋刃手斧
- 类型：物理 / 近战
- 定位：清怪 / 状态
- 初始品阶：紫

## 🧱 基础属性
- 攻击力：中
- 攻速：中
- 范围：中
- 暴击率：中
- 击退：中
- 状态附着概率：中

## 🧪 状态附带
- 状态1：灼烧｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：完成一次近战挥砍。
- 触发效果：向前释放回旋刃，飞出和返回各造成一次伤害并尝试附加灼烧。
- 状态交互：灼烧可与中毒触发爆裂，也可作为过载前置之一。
- Build 联动：适合搭配中毒、扩散和聚怪效果提高灼烧覆盖。
- 限制条件：回旋刃轨迹固定，玩家站位错误时收益明显下降。

## 🔧 合成 / 升阶
- 蓝 → 紫：基础轨迹延长。
- 紫 → 粉：回程命中灼烧目标时提高伤害。
- 粉 → 红：回旋刃可穿过首个敌人。
- 红 → 金：命中 4 个以上灼烧目标时产生一次小型火环。

## 🧬 状态升阶
- 状态成长方式：灼烧状态通过重复状态升阶。
- 叠加变化：灼烧层数提高持续伤害和后续反应稳定性。
- 与状态系统的关系：灼烧作为输出类状态，不应单独支撑通关，需要与中毒、感电或标记联动。

## 🧠 构筑方向
- 适合流派：灼烧扩散流 / 清怪状态流。
- 推荐搭配状态：灼烧、中毒、扩散。
- 推荐搭配武器类型：范围武器、聚怪武器、DOT 触发武器。
- 不适合搭配：纯单体精准武器。

## ⚖️ 优缺点
- 优势：清怪覆盖好，能自然建立灼烧前置。
- 劣势：贴身风险高，单体爆发不足。

## 🧩 平衡性备注
- 强度来源：往返两段命中与灼烧覆盖。
- 限制方式：轨迹、站位和中等攻速。
- 弱势场景：高机动单体、狭窄绕行地形。
