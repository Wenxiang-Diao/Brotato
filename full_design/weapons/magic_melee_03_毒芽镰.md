# 毒芽镰

## 📌 基本信息
- 武器名称：毒芽镰
- 类型：法术 / 近战
- 定位：持续 / 扩散
- 初始品阶：紫

## 🧱 基础属性
- 攻击力：中
- 攻速：中
- 范围：中
- 暴击率：低
- 击退：低
- 状态附着概率：高

## 🧪 状态附带
- 状态1：中毒｜蓝
- 状态2：扩散｜蓝
- 状态3：无

## 🔗 核心机制
- 触发条件：击杀带有中毒的敌人。
- 触发效果：向周围敌人传播一层中毒。
- 状态交互：中毒 + 扩散形成持续流基础，中毒也可与灼烧触发爆裂。
- Build 联动：适合搭配灼烧、聚怪和持续伤害增益。
- 限制条件：需要击杀触发，面对高血单体启动慢。

## 🔧 合成 / 升阶
- 蓝 → 紫：提高中毒附着概率。
- 紫 → 粉：传播范围增加。
- 粉 → 红：传播时有概率刷新中毒持续时间。
- 红 → 金：中毒目标死亡时生成短暂毒芽区域。

## 🧬 状态升阶
- 状态成长方式：中毒通过重复状态升阶，扩散随武器品阶强化范围。
- 叠加变化：中毒层数提高持续伤害，传播后更容易形成连锁感染。
- 与状态系统的关系：扩散只传播合法层数，不突破单状态上限。

## 🧠 构筑方向
- 适合流派：中毒扩散流 / 持续清怪流。
- 推荐搭配状态：中毒、扩散、灼烧。
- 推荐搭配武器类型：火焰爆发、聚怪、范围补刀武器。
- 不适合搭配：纯 Boss 单体构筑。

## ⚖️ 优缺点
- 优势：小怪越多越强，状态传播效率高。
- 劣势：启动依赖击杀，对单体较弱。

## 🧩 平衡性备注
- 强度来源：中毒击杀后的传播链。
- 限制方式：击杀触发和传播范围。
- 弱势场景：精英房、Boss 房、低怪密度节点。
