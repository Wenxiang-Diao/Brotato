# 裂片弹弓

## 📌 基本信息
- 武器名称：裂片弹弓
- 类型：物理 / 远程
- 定位：清怪 / 破甲
- 初始品阶：紫

## 🧱 基础属性
- 攻击力：中
- 攻速：中
- 范围：中
- 暴击率：低
- 击退：中
- 状态附着概率：中

## 🧪 状态附带
- 状态1：破甲｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：主弹丸命中敌人。
- 触发效果：分裂出 3 枚碎片，碎片命中破甲目标时可以穿透一次。
- 状态交互：破甲 + 高频碎片攻击形成穿透 DPS 提升。
- Build 联动：适合搭配聚怪、破甲层数提升和标记目标牵引。
- 限制条件：碎片角度固定，敌人过于分散时命中率下降。

## 🔧 合成 / 升阶
- 蓝 → 紫：碎片数量增加。
- 紫 → 粉：碎片对破甲目标伤害提高。
- 粉 → 红：主弹丸可对首个目标附加额外破甲概率。
- 红 → 金：穿透破甲目标后碎片有低概率再次分裂。

## 🧬 状态升阶
- 状态成长方式：破甲状态通过重复状态升阶。
- 叠加变化：破甲层数提高后，碎片穿透收益更稳定。
- 与状态系统的关系：碎片攻击不能突破状态层数上限，只提高触发频率。

## 🧠 构筑方向
- 适合流派：破甲穿透流 / 密集清怪流。
- 推荐搭配状态：破甲、吸附、标记。
- 推荐搭配武器类型：聚怪武器、高频射击、范围补伤武器。
- 不适合搭配：依赖敌人远离的风筝控制武器。

## ⚖️ 优缺点
- 优势：对密集敌人效率高，破甲后成长明显。
- 劣势：需要聚怪或站位，单体表现普通。

## 🧩 平衡性备注
- 强度来源：碎片数量和破甲穿透。
- 限制方式：固定散射角与低单体倍率。
- 弱势场景：分散远程怪、少量高血目标。
