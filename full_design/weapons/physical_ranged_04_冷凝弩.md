# 冷凝弩

## 📌 基本信息
- 武器名称：冷凝弩
- 类型：物理 / 远程
- 定位：控制 / 单体
- 初始品阶：紫

## 🧱 基础属性
- 攻击力：高
- 攻速：慢
- 范围：远
- 暴击率：中
- 击退：中
- 状态附着概率：高

## 🧪 状态附带
- 状态1：冻结｜蓝
- 状态2：无
- 状态3：无

## 🔗 核心机制
- 触发条件：蓄力完成后发射冷凝箭。
- 触发效果：箭矢飞行距离越远，冻结附着概率越高。
- 状态交互：冻结可与重击、爆发攻击形成破碎。
- Build 联动：适合搭配板砖、寒镜匕首等冻结后爆发武器。
- 限制条件：蓄力期间移动速度降低，近距离射击收益较低。

## 🔧 合成 / 升阶
- 蓝 → 紫：缩短蓄力时间。
- 紫 → 粉：远距离命中附加更高冻结层数。
- 粉 → 红：命中冻结目标时提高暴击伤害。
- 红 → 金：满蓄力击杀冻结目标时产生小范围冰爆。

## 🧬 状态升阶
- 状态成长方式：冻结状态通过重复状态升阶。
- 叠加变化：冻结层数提高控制稳定性和破碎窗口。
- 与状态系统的关系：Boss 控制效果削弱，但冻结状态仍可作为反应标记。

## 🧠 构筑方向
- 适合流派：冻结控制流 / Boss 压制流。
- 推荐搭配状态：冻结、破甲、标记。
- 推荐搭配武器类型：重击近战、单体爆发、区域冰冻武器。
- 不适合搭配：需要持续高速移动的贴脸风险武器。

## ⚖️ 优缺点
- 优势：远程控制稳定，单体压制强。
- 劣势：攻速慢，蓄力惩罚明显。

## 🧩 平衡性备注
- 强度来源：远距离冻结与高基础伤害。
- 限制方式：蓄力、低攻速和近距离收益衰减。
- 弱势场景：近身包围、大量高速小怪。
