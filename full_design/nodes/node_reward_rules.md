# 节点奖励规则

---

## 1. 奖励设计目标

节点奖励需要推动玩家构筑，而不是只提高数值。

奖励应服务以下目标：

- 让武器体系逐渐成型；
- 让状态反应逐渐清晰；
- 让角色技能有发挥空间；
- 让 debuff 风险有修正机会；
- 让水晶系统提供长期追求。

---

## 2. 奖励类型

| 奖励类型 | 作用 |
|---|---|
| 经验 | 提升等级，触发升级抽奖 |
| 金币 | 商店购买与修正资源 |
| 武器 | 直接改变攻击方式 |
| 武器升阶材料 | 推进武器品质 |
| buff | 强化 build 方向 |
| debuff | 风险代价或特殊构筑组件 |
| 状态强化 | 强化灼烧、中毒、冻结、感电等状态 |
| 水晶碎片 | 局外成长资源 |
| 羁绊点 | 角色长期成长与剧情解锁 |
| 图鉴素材 | 长期收集与成就系统 |

---

## 3. 奖励品质

| 品质 | 颜色 | 说明 |
|---|---|---|
| 普通 | 蓝 | 基础成长，低风险 |
| 稀有 | 紫 | 明显改变 build |
| 史诗 | 粉 | 强机制组件，可能带条件 |
| 传说 | 红 | 高影响奖励，通常绑定风险 |
| 神话 | 金 | 极低概率，适合 Boss / 隐藏节点 |

---

## 4. 各节点奖励表

| 节点 | 主要奖励 | 次要奖励 | 高稀有奖励 |
|---|---|---|---|
| 普通战斗 | 经验、金币 | 低品质 buff | 状态强化 |
| 精英战斗 | 高品质武器、稀有 buff | 水晶碎片 | 红色机制组件 |
| Boss | 章节奖励、水晶碎片 | 高品质武器 | 角色羁绊 / 解锁 |
| 事件 | 特殊选项 | buff / debuff 修正 | 隐藏构筑奖励 |
| 商店 | 可购买奖励 | debuff 修正 | 稀有商品 |
| 抽奖 | 三选一构筑奖励 | 重抽机会 | 高光奖励 |
| 休息 | 恢复 / 净化 | 精炼 | 下一抽奖加权 |
| 宝藏 / 水晶 | 水晶碎片 | 图鉴素材 | 金色局外资源 |
| 风险 | 高品质奖励 | 大量资源 | 特殊诅咒构筑 |

---

## 5. 风险收益规则

### 5.1 高收益奖励必须绑定条件

| 奖励强度 | 代价类型 |
|---|---|
| 小幅收益 | 无代价或轻微金币消耗 |
| 中等收益 | 血量、金币、放弃其他奖励 |
| 高收益 | 获得 debuff 或下一战强化敌人 |
| 极高收益 | 长期 debuff、Boss 强化或路线锁定 |

---

### 5.2 高光奖励示例

| 奖励 | 代价 |
|---|---|
| 获得紫色武器 | 下一场战斗敌人生命 +15% |
| 感电链跳 +1 | 获得“导电体质”：受到感电伤害 +10% |
| 灼烧爆裂范围 +20% | 获得“易燃”：受到火焰伤害 +10% |
| 移除 2 个 debuff | 放弃本层全部金币和经验 |
| 金色水晶碎片 | 立即进入一场守卫战 |

---

## 6. 抽奖节点奖励池

### 6.1 基础三选一

抽奖节点提供 3 个选项：

1. 武器 / 武器升阶；
2. 状态强化；
3. buff 或 build 组件。

### 6.2 权重影响因素

| 因素 | 影响 |
|---|---|
| 当前角色元素倾向 | 提高相关状态和武器权重 |
| 当前持有武器 | 提高同类升阶或互补状态权重 |
| 水晶成长 | 提高指定元素或构筑方向权重 |
| 当前 debuff | 提高修正或风险构筑选项权重 |
| 当前章节 | 提高高级奖励概率 |

## 8. Boss 奖励规则

Boss 奖励必须体现章节推进价值。

### 第 1 章 Boss

- 必给 1 个中品质 build 奖励；
- 解锁第 2 章奖励池；
- 给少量水晶碎片；
- 可选择一次 debuff 修正。

### 第 2 章 Boss

- 必给 1 个高品质武器或核心 buff；
- 给中量水晶碎片；
- 触发角色羁绊点；
- 提供最终章路线分支预览。

### 最终 Boss

- 给大量水晶碎片；
- 解锁图鉴和角色羁绊剧情；
- 根据通关 build 记录成就；
- 高难度下解锁新节点变体。
