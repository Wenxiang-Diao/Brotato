# 节点 / 关卡路线系统总览

> 模块：Node & Route System  
> 优先级：P0（核心局内流程系统）  
> 适用项目：《星链回响》（Echoes of Astra）  
> 当前版本：v1.0 Draft

---

## 1. 系统定位

节点 / 关卡路线系统用于承接一局 Roguelite 的主要推进节奏。

它不是单纯的“选下一关”，而是让玩家在每一局中持续做出路线决策：

- 要稳定成长，还是承担风险换取高收益；
- 要优先补武器，还是补状态反应；
- 要去商店修正 debuff，还是挑战精英获取稀有奖励；
- 要加快 build 成型，还是保存资源应对 Boss。

本系统服务于项目核心：

```text
战斗 → 获取资源 → 节点选择 → 抽奖 / 事件 / 商店 / 休息 → buff/debuff 调整 → 构筑推进 → Boss 检验
```

---

## 2. 设计目标

| 目标 | 说明 |
|---|---|
| 强化路线策略 | 玩家根据当前 build、血量、debuff 和资源选择路线 |
| 推动构筑成型 | 节点奖励直接影响武器、状态、buff 和元素反应 |
| 承接风险管理 | 高收益节点绑定 debuff、精英压力或资源代价 |
| 提供短局节奏 | 一局路线清晰，避免过长或过碎 |
| 支持长期扩展 | 后续可增加地图、章节、事件、Boss、节点变体 |

---

## 3. 一局基础结构

MVP 推荐采用 **3 章制 + 18 层节点**。

| 章节 | 层数 | 主题定位 | 主要目标 |
|---|---:|---|---|
| 第 1 章：星屑林地 | 1-6 | 教学与 build 起步 | 获取首批武器 / 状态 / buff |
| 第 2 章：晶脉废墟 | 7-12 | 风险与路线分化 | 形成核心 build，处理 debuff |
| 第 3 章：星骸回廊 | 13-18 | 高压检验 | 强化成熟 build，挑战最终 Boss |

关键层：

| 层数 | 推荐节点规则 |
|---:|---|
| 1 | 固定普通战斗节点 |
| 3 | 必出抽奖或事件节点之一 |
| 6 | 第 1 章 Boss |
| 9 | 必出精英 / 商店二选一分支 |
| 12 | 第 2 章 Boss |
| 15 | 高风险节点出现率提升 |
| 17 | 最终准备层，商店 / 休息 / 宝藏至少出现一个 |
| 18 | 最终 Boss |

## 4. 节点类型总览

| 节点 | 功能 | 主要奖励 | 主要风险 |
|---|---|---|---|
| 普通战斗 | 基础推进 | 经验、金币、低中品质奖励 | 战斗消耗 |
| 精英战斗 | 高风险挑战 | 高品质武器、稀有 buff、水晶碎片 | 高强度怪物、精英词缀 |
| Boss | 章节检验 | 大量资源、核心解锁、章节推进 | 高压战斗、阶段机制 |
| 事件 | 分支选择 | 特殊 buff、移除 debuff、隐藏奖励 | 随机代价、未知结果 |
| 商店 | 资源转换 | 购买武器、道具、修正 debuff | 消耗金币 |
| 抽奖 | 构筑推进 | 三选一强化、状态强化 | 可能附带 debuff |
| 休息 | 修正恢复 | 回血、移除 debuff、调整 build | 放弃成长收益 |
| 宝藏 / 水晶 | 局外与局内成长 | 水晶碎片、稀有资源 | 奖励偏长期，短期战力较弱 |
| 风险节点 | 高收益换风险 | 高品质奖励、特殊构筑机会 | 强制 debuff、敌人强化 |

## 6. MVP 实现范围

MVP 第一版建议实现：

- 18 层路线结构；
- 7 类基础节点：普通战斗、精英、Boss、事件、商店、抽奖、休息；
- 12 个事件节点；
- 3 个章节 Boss；
- 基础路线生成算法；
- 基础奖励权重；
- 与 debuff 修正窗口的简单联动；
- 与水晶碎片的局外成长入口。

宝藏 / 水晶节点和风险节点可作为第一版扩展内容，也可以在 MVP 中以低频节点形式出现。
