# Boss 与章节路线结构

---

## 1. 章节总览

MVP 推荐 3 章结构。

| 章节 | 层数 | 主题 | Boss | 主要检验 |
|---|---:|---|---|---|
| 第 1 章 | 1-6 | 星屑林地 | 星骸巨像 | 基础输出与走位 |
| 第 2 章 | 7-12 | 晶脉废墟 | 雷鸣偶核 | 状态反应与控场 |
| 第 3 章 | 13-18 | 星骸回廊 | 腐耀花皇 | 完整 build 与风险管理 |

---

## 2. 第 1 章：星屑林地

### 主题目标

让玩家快速理解：

- 自动攻击；
- 主动技能；
- 状态附着；
- 基础节点路线；
- 第一次 Boss 检验。

### 推荐节点分布

| 层数 | 节点设计 |
|---:|---|
| 1 | 固定普通战斗 |
| 2 | 普通战斗 / 抽奖 |
| 3 | 事件 / 抽奖 / 普通战斗 |
| 4 | 普通战斗 / 商店 / 精英 |
| 5 | 休息 / 商店 / 抽奖 |
| 6 | Boss：星骸巨像 |

### Boss：星骸巨像

检验点：

- 玩家是否有稳定输出；
- 是否理解躲避大范围攻击；
- 是否有至少一种状态反应。

Boss 抗性：

- 控制类状态持续时间降低；
- 破甲有效；
- 灼烧、中毒、感电可正常触发但有层数衰减。

---

## 3. 第 2 章：晶脉废墟

### 主题目标

让玩家开始面对：

- 精英词缀；
- 状态抗性；
- 高风险节点；
- debuff 修正；
- 路线分化。

### 推荐节点分布

| 层数 | 节点设计 |
|---:|---|
| 7 | 普通战斗 / 抽奖 / 事件 |
| 8 | 普通战斗 / 精英 / 商店 |
| 9 | 精英 / 商店二选一分支 |
| 10 | 事件 / 风险 / 普通战斗 |
| 11 | 休息 / 抽奖 / 宝藏 |
| 12 | Boss：雷鸣偶核 |

### Boss：雷鸣偶核

检验点：

- 玩家是否有处理召唤物或装置的能力；
- 是否能承受连锁伤害压力；
- 是否能利用状态反应打断 Boss 节奏。

Boss 抗性：

- 感电伤害衰减，但可触发反应；
- 标记有效；
- 冻结仅造成短暂停滞；
- 破甲效果较好。

---

## 4. 第 3 章：星骸回廊

### 主题目标

最终章要求玩家完成：

- 成熟 build；
- debuff 管理；
- 高密度清怪；
- Boss 多阶段应对；
- 最终路线取舍。

### 推荐节点分布

| 层数 | 节点设计 |
|---:|---|
| 13 | 普通战斗 / 事件 / 抽奖 |
| 14 | 精英 / 普通战斗 / 宝藏 |
| 15 | 风险 / 精英 / 商店 |
| 16 | 抽奖 / 事件 / 普通战斗 |
| 17 | 休息 / 商店 / 宝藏 |
| 18 | Boss：腐耀花皇 |

### Boss：腐耀花皇

检验点：

- 玩家是否能处理持续伤害和场地压力；
- 是否能清理召唤物；
- 是否能在 debuff 压力下维持输出；
- 是否拥有完整反应链。

Boss 抗性：

- 中毒抗性较高；
- 灼烧可触发爆裂；
- 冻结只触发减速；
- 感电和标记对召唤物群有效。

---

## 5. Boss 前准备层规则

每个 Boss 前一层必须至少包含一个准备节点：

| 准备节点 | 作用 |
|---|---|
| 休息 | 恢复生命或移除 debuff |
| 商店 | 购买补给或修正 build |
| 抽奖 | 获取最终补强 |
| 宝藏 / 水晶 | 获取长期收益或特殊资源 |

如果路线生成没有准备节点，系统强制替换一个普通战斗节点。

---

## 6. Boss 奖励和章节推进

### 第 1 章 Boss 奖励

- 1 个中品质构筑奖励；
- 少量水晶碎片；
- 1 次 debuff 修正机会；
- 解锁第 2 章奖励池。

### 第 2 章 Boss 奖励

- 1 个高品质武器 / buff；
- 中量水晶碎片；
- 角色羁绊点；
- 最终章路线预览。

### 最终 Boss 奖励

- 大量水晶碎片；
- 角色羁绊剧情推进；
- Boss 图鉴解锁；
- 通关 build 记录；
- 高难度 / Endless 解锁条件。
