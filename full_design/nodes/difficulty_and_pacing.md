# 难度递增与关卡节奏

---

## 1. 节奏目标

本项目一局推荐控制在 20-35 分钟。

节奏曲线：

```text
建立方向 → 第一次检验 → 路线分化 → 第二次检验 → 高压成型 → 最终 Boss
```

---

## 2. 章节节奏

### 第 1 章：星屑林地

| 层数 | 体验目标 |
|---:|---|
| 1 | 熟悉移动、自动攻击和基础状态 |
| 2-3 | 获得第一批武器 / buff / 状态强化 |
| 4 | 第一次路线分支选择 |
| 5 | Boss 前准备 |
| 6 | 第 1 章 Boss，检验基础 build |

难度特征：

- 怪物速度低；
- 精英数量少；
- debuff 压力低；
- 事件代价较轻。

---

### 第 2 章：晶脉废墟

| 层数 | 体验目标 |
|---:|---|
| 7-8 | 提高怪物密度，强化状态需求 |
| 9 | 精英 / 商店二选一路线 |
| 10-11 | build 分化，出现风险节点 |
| 12 | 第 2 章 Boss，检验核心反应链 |

难度特征：

- 敌人开始出现抗性；
- 精英词缀出现；
- 高收益选项附带 debuff；
- 玩家需要开始主动修正风险。

---

### 第 3 章：星骸回廊

| 层数 | 体验目标 |
|---:|---|
| 13-14 | 高密度怪潮，要求 AOE 或控制能力 |
| 15 | 风险节点权重提升 |
| 16 | 最终 build 补强 |
| 17 | 最终准备层 |
| 18 | 最终 Boss，检验完整构筑 |

难度特征：

- 怪物组合复杂；
- 精英词缀组合出现；
- Boss 对控制有抗性；
- debuff 数量可能成为主要压力。

---

## 3. 难度参数递增

| 参数 | 第 1 章 | 第 2 章 | 第 3 章 |
|---|---:|---:|---:|
| 普通怪生命倍率 | 1.0 | 1.6 | 2.4 |
| 普通怪伤害倍率 | 1.0 | 1.4 | 2.0 |
| 怪物密度 | 低 | 中 | 高 |
| 精英出现率 | 低 | 中 | 高 |
| 状态抗性 | 低 | 中 | 中高 |
| debuff 压力 | 低 | 中 | 高 |
| 奖励品质 | 蓝 / 紫 | 紫 / 粉 | 粉 / 红 |

---

## 4. 怪物投放规则

### 普通战斗节点

| 章节 | 投放组合 |
|---|---|
| 第 1 章 | 普通怪 + 少量远程怪 |
| 第 2 章 | 普通怪 + 机制怪 + 远程怪 |
| 第 3 章 | 高密度普通怪 + 特殊机制怪 + 小精英 |

---

### 精英节点

| 章节 | 精英配置 |
|---|---|
| 第 1 章 | 单精英 + 少量小怪 |
| 第 2 章 | 单精英 + 机制怪群 |
| 第 3 章 | 双阶段精英或精英 + 特殊装置 |

---

## 5. 状态抗性递增

为了防止单状态通关，敌人会逐渐获得轻度抗性。

| 敌人类型 | 状态适配 |
|---|---|
| 普通怪 | 易叠状态 |
| 精英怪 | 某一状态抗性较高 |
| Boss | 控制衰减，但不完全免疫 |
| 特殊怪 | 可能反制某类状态 |

控制类状态建议：

- 普通怪：100% 效果；
- 精英怪：60%-80% 效果；
- Boss：25%-40% 效果，但可触发破碎等交互。

---

## 6. Debuff 压力曲线

| 阶段 | Debuff 设计 |
|---|---|
| 第 1 章 | 少量、教学性质 |
| 第 2 章 | 奖励绑定 debuff 开始常见 |
| 第 3 章 | debuff 成为路线决策核心 |

每 5 级修正窗口建议与节点系统联动：

- 如果玩家即将进入 Boss 层，可优先出现防御 / 净化选项；
- 如果玩家 debuff 数量过多，休息或商店节点权重轻微提升；
- 风险型角色可获得利用 debuff 的特殊选项。

---

## 7. Boss 前准备规则

Boss 前一层必须至少出现以下节点之一：

- 商店；
- 休息；
- 抽奖；
- 宝藏 / 水晶。

如果路线生成结果没有准备节点，需要强制替换一个普通战斗节点。

---

## 8. 失败体验控制

为了避免玩家觉得失败完全来自随机：

- 第 1 章不得出现高压风险节点；
- 第 1 章 Boss 前必须至少获得 2 次构筑奖励；
- 连续两次事件不得都是纯负面；
- 精英节点进入前显示词缀；
- Boss 前提供至少一个修正机会。
