# 🧬 水晶系统与角色羁绊联动

---

## 1. 设计目标

角色羁绊与水晶系统结合，用于增强角色长期成长，但不能让角色变成数值皮肤。

目标：

- 让每个角色拥有长期培养价值。
- 让角色玩法与水晶分支产生差异。
- 解锁剧情、语音、挑战和轻量机制修正。
- 保持角色之间的 build 开放性。

---

## 2. 羁绊资源

资源名：**羁绊露滴**

获得方式：

- 使用该角色完成一局
- 使用该角色击败 Boss
- 使用该角色完成专属挑战
- 使用该角色触发指定元素反应次数
- 解锁角色剧情节点

---

## 3. 羁绊等级

| 等级 | 解锁内容 |
|---|---|
| Lv.1 | 角色档案、基础语音、推荐 build 提示 |
| Lv.2 | 角色专属水晶小节点 |
| Lv.3 | 角色剧情 1、专属挑战 |
| Lv.4 | 专属修正选项进入奖励池 |
| Lv.5 | 角色终点羁绊节点 |

---

## 4. 角色专属水晶节点原则

角色节点应提供“玩法微调”，不要提供无脑强度。

推荐：

- 初始奖励倾向微调
- 专属状态反应提示
- 特定风险玩法缓冲
- 特定武器标签权重轻微提升
- 特殊剧情事件解锁

避免：

- 永久大幅加伤害
- 永久减少全部冷却
- 永久免疫 Debuff
- 直接赠送高级武器

---

## 5. 四个初始角色联动示例

### 5.1 露米 · 星辉学徒

定位：新手均衡型。

推荐联动：星核 + 结晶。

羁绊节点：

| 节点 | 效果 |
|---|---|
| 星辉笔记 | 每局第一次升级奖励中，更容易出现通用型奖励 |
| 稳定练习 | 第一次修正窗口额外显示 1 个说明提示 |
| 学徒共鸣 | 同时拥有 2 种不同状态奖励时，获得少量星尘加成 |

### 5.2 岚音 · 雷纹偶像

定位：状态连锁型。

推荐联动：导能。

羁绊节点：

| 节点 | 效果 |
|---|---|
| 舞台导线 | 感电 + 标记奖励权重略微提升 |
| 连锁节拍 | 每章首次触发 5 次以上链跳时，获得导能晶片 |
| 雷纹谢幕 | 终极技能击杀标记敌人时，额外积累少量羁绊露滴 |

### 5.3 烬牙 · 赤晶狂徒

定位：风险收益型。

推荐联动：灼焰 + 裂隙残片。

羁绊节点：

| 节点 | 效果 |
|---|---|
| 烬火赌约 | 选择风险奖励后，灼焰奖励权重短暂提升 |
| 赤晶代价 | 每局首次危险 Debuff 可获得额外裂隙残片，但无法立即移除 |
| 狂徒余烬 | 低血量通关节点时，获得少量灼焰晶片 |

### 5.4 澪珀 · 凝流守望者

定位：控制 / 生存型。

推荐联动：凝流 + 生息。

羁绊节点：

| 节点 | 效果 |
|---|---|
| 静水护符 | 冻结奖励权重略微提升 |
| 守望回声 | 控制精英敌人后获得少量护盾，有冷却 |
| 凝流誓约 | 使用休息节点时，有概率出现凝流专属修正 |

## 6. 角色挑战示例

| 角色 | 挑战 | 奖励 |
|---|---|---|
| 露米 | 使用 3 种不同状态完成第一章 | 星核节点解锁材料 |
| 岚音 | 单局触发 100 次感电链跳 | 导能晶片 + 羁绊露滴 |
| 烬牙 | 携带 3 个 Debuff 击败精英 | 裂隙残片 + 灼焰晶片 |
| 澪珀 | 冻结精英 10 次并通关节点 | 凝流晶片 + 羁绊露滴 |

---

## 7. 平衡限制

- 羁绊节点不能强制绑定唯一 build。
- 羁绊奖励更多是倾向和提示，不是巨大数值。
- 角色专属节点应在角色使用时才生效，避免所有角色共享导致堆叠失控。
- 角色羁绊不应替代水晶分支主成长。
