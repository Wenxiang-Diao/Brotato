# 🌳 水晶 / 局外成长系统总览（Crystal & Meta Progression）

> 系统定位：局外长期成长核心  
> 优先级：P0  
> 依赖系统：抽奖 / 奖励、Debuff、元素 / 状态、角色、节点路线、怪物、商店

---

## 1. 系统目标

水晶系统用于解决三件事：

1. **提供长期成长目标**：玩家每局结束后都能获得可积累的成长资源。
2. **引导构筑方向**：水晶不直接替玩家完成 build，而是影响抽奖权重、风险承受能力和元素倾向。
3. **增强重复游玩动力**：通过分支选择、角色羁绊、特殊节点解锁，让不同游玩路线逐渐形成差异。

本系统不是单纯的永久属性加成树。它的核心是：

> 通过局外选择改变局内“可能性分布”，而不是直接替代局内决策。

---

## 2. 核心循环

```text
完成一局 / 失败结算
→ 获得星尘、元素晶片、古树芽核
→ 升级水晶节点
→ 改变抽奖权重 / 元素倾向 / 风险修正能力
→ 进入下一局
→ 形成更明确的构筑路线
```

## 3. 六大水晶分支

| 分支 | 主题 | 对应玩法 | 主要影响 |
|---|---|---|---|
| 导能水晶 | 感电 / 标记 / 连锁 | 高频触发、连锁清怪 | 感电、标记、链跳、状态传播 |
| 生息水晶 | 中毒 / 吸血 / 续航 | 持续伤害、恢复、生存 | DOT、吸血、回复、持久战 |
| 凝流水晶 | 冻结 / 控制 / 防守 | 控制、减速、空间管理 | 冻结、护盾、控制收益 |
| 灼焰水晶 | 灼烧 / 爆裂 / 高风险 | 爆发、范围伤害、风险收益 | 灼烧、爆裂、过载、危险奖励 |
| 结晶水晶 | 破甲 / 防御 / 稳定 | 抗压、破甲、稳健成长 | 破甲、防御、修正窗口 |
| 星核水晶 | 通用 / 路线 / 奖励 | 全局系统修正 | 抽奖、重抽、节点、资源 |

## 5. MVP 建议

MVP 阶段不需要一次性实现完整水晶树。建议先实现：

- 6 个分支
- 每个分支 5 个节点
- 每个节点最多 3 级
- 1 个重置功能
- 1 套基础资源结算
- 抽奖权重、Debuff 修正、元素倾向三类核心影响

完整版再扩展到每分支 8~10 个节点。
